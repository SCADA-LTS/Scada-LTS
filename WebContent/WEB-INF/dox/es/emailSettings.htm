La configuración del correo electrónico se utiliza para todos los envíos de Scada-LTS.

El SMTP host es el nombre del servidor en el que se ejecuta el MTA (Agente de Transferencia de Correo). Si el servidor requiere autenticación, marque la casilla autorización de uso y rellene los campos nombre de usuario y contraseña. Haga clic en la casilla Habilitar TLS si el servidor lo requiere.

El campo dirección es la dirección de envío utilizada para todos los correos. El campo nombre es un campo opcional para especificar un nombre propio o un alias para el remitente.

Utilice el cuadro de selección tipo de contenido que quiere ver en los correos. Por defecto HTML y text está seleccionado. Nota: El envío de informes programados por correo siempre es sólo HTML debido al contenido de estos correos.

Una vez definidos los parámetros, puede probar su configuración haciendo clic en el icono . Esto intentará enviar un correo electrónico con su nueva configuración.
