Las configuraciones de correo son utilizadas para todas las actividades de correo electrónico en la aplicación.

El Servidor SMTP es el nombre del servidor donde está operando su MTA. Si el servidor requiere autentificación, active la casilla de Utilice autorización, y llene los campos correspondientes con el Nombre de Usuario y su Contraseña. Igualmente, si su servidor requiere seguridad en la capa de trasporte habilite la casilla Habilitar TLS.

La dirección 'De' es la dirección que será utilizada como dirección electrónica del remitente de cada correo enviado. El Nombre 'De' es un campo opcional que le permite especificar un alias—o un nombre 'personal'— para ser mostrado en el remitente del correo.

Utilice la caja de selección del Tipo de contenido para cambiar la el tipo de forma en la que se enviarán los correos. De forma predefinida se utiliza HTML y texto, por lo que se crea un MIME con partes con el formato "texto/html" y otras con "texto/texto". Cuando selecciona Sólo HTML el correo estará conformado por varias partes cuando tenga adjuntos o contenido en línea, aunque en los demás casos será de una única parte. En el caso que se eligiera Sólo Texto, el correo siempre estará conformado por una parte única. Nota: los correos que son programados para entregar los reportes siempre se enviarán como sólo HTML sin importar el contenido que tengan.

Cuando termine con las configuraciones de correo, puede probarlas haciendo clic en el icono . Esto intentará enviarle un correo electrónico de prueba a la dirección de correo definida utilizando las configuraciones realizadas.
