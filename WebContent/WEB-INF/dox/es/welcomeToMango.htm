Bienvenido al ScadaLTS

(Impulsado por Mango)

ScadaLTS es una Plataforma M2M de código abierto con capacidades que sólo podrían obtenerse por un costo por instalación de millones de dólares (si tiene todo). Todo este potencial es proporcionado vía internet desde un navegador por lo que no requiere instalación de software ni de plugins.

Por favor ingrese su "Nombre de Usuario" para iniciar la sesión. Si requiere una cuenta de usuario en este sistema u olvidó su contraseña, contacte al administrador del ScadaLTS para asistencia.

Si acaba de instalar el ScadaLTS, ingrese con nombre de usuario "admin" y contraseña "admin". Note que por razones de seguridad debería cambiar esta contraseña inmediatamente.
