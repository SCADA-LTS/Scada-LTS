Configuración

Cada fuente de datos requiere un Nombre, que puede contener cualquier descripción.

La dirección de la fuente es la dirección principal, o, la dirección de la fuente de datos a registrar y la la dirección del esclavo es la dirección del dispositivo remoto al que se quiere comunicar.

El período de votación RBE corresponde al intervalo de tiempo en el que el sistema hace requisiciones de clase 1, 2 y 3 (eventos) al dispositivo esclavo. El período de sondeo estático corresponde al intervalo de tiempo durante el cual el sistema hace solicitudes de clase 0 al y es relativo al período de sondeo RBE, por lo tanto, si el período RBE = 2s y el período de sondeo estático = 30, la fuente de datos requiere datos de clase 0 cada 60 segundos al dispositivo.
