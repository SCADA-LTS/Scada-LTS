El ID de la instancia es asignado automáticamente a su instancia Scada-LTS por los servidores de Home Scada-LTS, y no debe ser modificado. Se utiliza, entre otras cosas, para asignar derechos sobre la documentación. Cuando se actualiza a una versión comercial puede que sea necesario determinar este valor.

El tamaño de la base de datos es el espacio que ocupa actualmente la base de datos Scada-LTS. La mayor parte de este espacio está generalmente ocupado por instancias de estado, lecturas puntuales y eventos. Para minimizar el tamaño de la base de datos, es necesario aplicar reglas de purga muy precisas.

Haga clic en el icono para comprobar las nuevas versiones. Las versiones también pueden ser revisadas periódicamente. Si se dispone de una actualización, se mostrará una alerta informativa. Este panel también muestra la versión instalada de Scada-LTS.
