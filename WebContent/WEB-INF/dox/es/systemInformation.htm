El ID de la Instancia se asigna automáticamente a su instancia del Servidor Principal del ScadaBR, y no debería cambiarse. Este ID se utiliza para dar la permisología sobre la documentación y otras cosas. Cuando realice una actualización a una versión comercial del ScadaBR, Ud requerirá conocer este valor.

El Tamaño de la base de Datos muestra el uso de espacio de almacenamiento que actualmente ocupa la Base de Datos del ScadaBR. En típicamente la mayoría del espacio es ocupado por reportes, lectura de puntos, y eventos. Para minimizar su tamaño utilice la herramienta Purgar para cada elemento tan completo y extenso como requiera.

Haga clic en el icono para realizar una revisión de la disponibilidad de una nueva versión del ScadaBR. El sistema está configurado para que de forma periódica y automática haga esta revisión. Cuando hay una actualización disponible se mostrará una Alerta de Información sobre su disponibilidad. En este panel igualmente se muestra la versión del ScadaBR actualmente ejecutándose.
