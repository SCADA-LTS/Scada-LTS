Visión general

Se utiliza un motor de representación gráfica para proporcionar el estado histórico general de un punto a lo largo de un determinado período de tiempo o de una serie de valores. La información agregada que puede proporcionarse depende del tipo de datos de los valores que se han de rendir. Por lo tanto, sólo aquellos tipos de motor de representación que se aplican al tipo de datos del punto actual se enumeran en la selección Tipo.

Nota! La representación gráfica depende de datos registrados. Si su punto tiene un tipo de registro "No registrar", es posible que su herramienta de representación gráfica no se muestre como se esperaba. La posible excepción es el gráfico de la tabla, que puede utilizar valores de puntos almacenados en la memoria, pero sólo si esos valores están disponibles. Reiniciando el sistema se borran todos los puntos de valor de las cachés.

Tablas gráficas

Este motor de representación muestra un número dado (el Límite) de los valores históricos más recientes para el punto. Los valores se muestran en una tabla que proporciona el valor (representado por el texto), el valor temporal y cualquier anotación aplicable. Este cuadro está disponible para todos los tipos de datos.

Estadísticas gráficas

Este gráfico muestra una tabla de información estadística calculada a partir de los valores de puntos históricos para el período de tiempo dado Período de tiempo. Los datos reales que se muestran dependen del tipo de datos del punto. Este gráfico está disponible para todos los tipos de datos.

Binario y multiestado: Estos tipos de datos producen una lista de estadísticas de "inicio y tiempo de ejecución" para cada estado que el punto ha experimentado durante el período de tiempo. Cada elemento de "inicio y tiempo de ejecución" proporciona el número de veces que se introdujo el estado ("inicios"), el tiempo total en que el punto estuvo en el estado ("tiempo de ejecución") y el porcentaje de la duración total del período que representa el tiempo de ejecución.

Digital : Los puntos numéricos producen los siguientes valores estadísticos para el período de tiempo: valor mínimo y momento de ocurrencia, valor máximo y momento de ocurrencia, media y suma (adecuado para el recuento de pulsos)..

Alfanumérico : Se indica el número de veces que el punto ha cambiado durante el período.

Gráficos de imagen

Este gráfico muestra un gráfico de barras lineales de los valores de los puntos en el período dado. Está disponible para puntos con tipos de datos numéricos o multiestatales.
