El ScadaBR actualmente permite el uso de dos Manejadores de Sistemas de Bases de Datos (DBMS): Derby y MySQL.

La Base de Datos predefinida del sistema es Derby. En el caso que requiera realizar cambios sobre la configuración inicial, debe modificar el archivo ubicado en "ScadaBR*/WEB-INF/classes/env.properties" y en "ScadaBR*/WEB-INF/classes/env.properties.derby".

Para utilizar MySQL, es requerido que configure la información para su correcto funcionamiento. En este caso debe modificar el contenido de "ScadaBR*/WEB-INF/classes/env.properties.mysql". Básicamente debe configurar los siguientes parámetros:

db.username: Nombre de Usuario.

db.password: Contraseña del Usuario.

Para este cambio es requerido elegir la opción MySQL y posteriormente hacer clic en el icono de guardar. Finalmente debe reiniciar el servidor de aplicación web (p.e: Tomcat).

ScadaBR*: La ubicación del servidor de aplicación web del ScadaBR (p.e: "Pasta_Tomcat/webapps/ScadaBR").
