Visión general

Esta fuente de datos se utiliza para reunir información genérica de los sitios web disponibles en la Internet, o de instalaciones internas disponibles en una intranet. Mediante mecanismos similares a los de un navegador web, la fuente de datos solicita información a los dispositivos habilitados para HTTP y luego utiliza expresiones regulares para extraer la información deseada. Esta es una fuente de datos de encuestas.

Configuración

Cada fuente de datos requiere un Nombre, que puede ser cualquier descripción. El Período de Actualización determina la frecuencia de la petición HTTP. La URL es la dirección de la página que necesita ser analizada. Usa el icono para probar la URL. (Se abre en una nueva ventana o pestaña del navegador).

Los campos Tiempo de espera y Reintenta determinan cuánto tiempo hay que esperar a que el servidor web devuelva la página, y cuántas veces hay que reintentar la conexión en caso de fallo. Si se agotan todos los intentos, se desencadena un evento de excepción de la fuente de datos.
