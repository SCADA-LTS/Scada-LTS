Esta página muestra los comandos para cambiar un punto. El icono es un enlace de sólo lectura a la página de detalles del punto; es visible sólo para los puntos guardados.

El nombre del punto es obligatorio, y puede contener cualquier texto. Este nombre definirá visualmente este punto para todo el sistema.

Se muestra el nombre de la fuente de datos. El icono te redirigirá directamente a la página de edición de la fuente de datos que contiene este punto.
