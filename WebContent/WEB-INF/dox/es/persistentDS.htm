Descripción General

Este tipo de fuente de datos es utilizada para recolectar información genérica de Pachube. La data es organizada en Pachube en feeds, los cuales son una colección de flujos de datos. Desde los mapas de flujo de datos a un punto en Mango, la definición de un punto Pachube requiere especificar que lo alimenta (feed) y su ID de flujo. Entonces, esta es una fuente de datos de sondeo. Desde los flujos de datos de los mapas hasta el punto en Mango. Desde los mapas de un flujo de datos a un punto de Mango, un punto Pachube logra especificando su alimentación y el ID de flujo. Entonces se trata de una fuente de datos de sondeo.

Configuración

Cada fuente de datos requiere un Nombre, el cual puede ser su descripción. El período de actualización establece cuan a menudo se realizan para realizar la solicitud de HTTP del Pachube. Note que - dependiendo de su cuenta en Pachube - API pueden aplicar algunas restricciones. Vea http://community.pachube.com/api#rate_limit Para detalles.

Se requiere de la clave API para acceder a datos privados. Puede definirla en el área de "mis configuraciones" cuando inicie sesión en su cuenta Pachube. Los campos tiempo de espera y reintentos determina cuento tiempo se esperará por el regreso de la data del servidor, y cuantas veces se intentará realizar la conexión cuando el evento falla. Cuando todos los reintentos se agotan, se generará un evento de excepción por la falla de la fuente de datos.
