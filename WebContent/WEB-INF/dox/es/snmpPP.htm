OID representa el identificador del objeto que se solicitará en el huésped o que se emparejará en las trampas que se reciban. El tipo de datos Datos es la representación Scada-LTS del valor. Los datos recibidos se convierten automáticamente en el tipo de datos requerido. El valor Binario 0 es el valor a igualar para que un punto binario reciba el valor 0. El valor Set type es el tipo de datos SNMP que se usará para establecer los valores en el host. Un valor de "(no configurable)" indica que el punto es de sólo lectura. El campo Survey determina si se incluye el valor en las consultas GET en cada ciclo de votación, o si los valores se reciben sólo mediante trampas.
