Visión general

El formato permite formatear un valor crudo para hacerlo más explícito para el usuario. Las opciones de formato dependen del tipo de valor que se muestre. Por lo tanto, sólo los formatos aplicables al valor de un punto se mostrarán en la lista desplegable del campo Tipo.

Binario

El tipo Binario permite al usuario definir una etiqueta para representar cada uno de los dos estados binarios(0 y 1). Todas las etiquetas están permitidas. También puedes definir un color para cada estado. Los ejemplos típicos de etiquetas son apagado/encendido, encendido/apagado, o verdadero/falso.

El formato por defecto de un valor binario es Simple. Muestra el valor bruto, 0 o 1.

Digital

La opción Analógico permite al usuario definir un Formato que especificará por ejemplo el número de decimales a mostrar, y la presencia de mil separadores. (Véase la documentación "Número de formato"). También se puede añadir un Suffix para mostrar las unidades del valor. Los sufijos pueden contener entidades HTML como °(escrito '&deg;') para la visualización de la página. Ver http://www.w3.org/TR/REC-html40/sgml/entities.html

El tipo Simple es el formato por defecto para los valores numéricos. Muestra el valor sin modificaciones.

El tipo Rango permite definir un formato de número Formato (Ver la documentación "Formato de número"), y textos de su elección para representar un valor según rangos de valores. A cada rango se le puede asignar un color distinto. El campo de formato se aplicará si el valor no corresponde a ningún rango definido. Por ejemplo, puedes clasificar la temperatura en "fría", "normal" y "cálida", y asignar un color para cada rango.

El tipo Hora permite al usuario definir un Formato (Ver la documentación "formato de fecha/hora"). Un formato de fecha típico sería "MM dd, yyyy". Un tiempo sería "HH:mm:ss". Los formatos de fecha y hora pueden combinarse si es necesario. También es posible convertir los tiempos en milisegundos. Se usarán exponentes de potencia 10 para estas conversiones. Por ejemplo, para convertir los segundos introduzca el parámetro 3, para los microsegundos introduzca -3. Si el valor ya está en milisegundos, introduzca 0. Se supone que los valores de tiempo comienzan en 0.1 de enero de 1970 a medianoche

Multiestado

La opción Simple es el formato por defecto para los valores multiestado. Esto muestra el valor bruto sin cambios.

la opción Multiestado permite asociar un texto, elegido arbitrariamente, al valor de un estado para representar los diferentes estados del punto. Cada estado también puede tomar un color diferente. Si un informe no tiene una etiqueta asociada, se mostrará el valor. Ejemplos típicos de etiquetas encendidas/apagadas, o apagadas/calientes/frías.

Alfanumérico

La opción Simple es el único formato alfanumérico. Se mostrarán los valores brutos, opcionalmente acompañados de una opción Suffix..
