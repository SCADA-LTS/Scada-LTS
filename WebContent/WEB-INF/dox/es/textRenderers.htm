Descripción General

El propósito del generador de texto es el de convertir un valor en un formato que sea más reconocible e informativo para quien lo interprete. Las formas para generar un conjunto de valores, dependerán del tipo de dato a ser transformado. Entonces, para cada punto se listan los "Tipos" en la selección de tipos de representación en texto que le apliquen según su tipo de datos.

Binario

Una representación para un binario permite definir etiquetas de texto que representará cada uno de los estados de un punto (p.e. 0 y 1). Puede asignarles cualquier etiqueta, al igual que asignarle un color que distinga cada estado. Ejemplos típicos de etiquetas son: encendido/apagado, off/on, apagado/encendido, o cierto/falso, pudiéndoles aplicar cualquier texto que desee.

De forma predefinida su será Plana, mostrando los valores crudos 0 y 1.

Numérico

La opción de representación para puntos Analógicos le permite definir el Formato a utilizar para determinar aspectos, tales como: la cantidad de lugares decimales a mostrar, el delimitador de miles, entre otros. (Vea la documentación de "Formatos Numéricos"). Adicionalmente puede agregarle un Sufijo al valor a mostrar de forma de incluirle sus unidades. Los sufijos pueden contener entradas en HTML tales como ° (se escriben como '&deg;') para mostrarse en páginas HTML. Vea http://www.w3.org/TR/REC-html40/sgml/entities.html para más información de entradas HTML.

De forma predefinida su representación será Plana, mostrando los valores inalterados del punto.

Al utilizar la opción Rango permite definir un Formato numérico (vea la documentación de "Formatos Numéricos" para más información), de esta forma podrá representarlo tanto en números arbitrarios, como en representaciones de rangos de texto o números desde/hasta. Cada uno de esos rangos puede están identificado con colores distintivos. Por ejemplo pudiera definir rangos denominados "frío", "normal", y "caliente" para definir los rangos de valores de un punto de temperatura, cada uno de estos identificados con colores apropiados. En el caso que un valor no esté incluido en ninguno de los rangos definidos, este valor se mostrará en formato numérico.

La opción Fecha/Hora permite configurar un Formato (vea la documentación "Formato fecha/hora" para más información). Un formato típico de fecha pudiera ser "MMM d, aaaa", y una de hora "HH:mm:ss". Los formatos de fecha y hora pueden combinarse cuando lo requiera. Igualmente tiene la posibilidad de utilizar una conversión de exponente en milisegundos. Esta es la posibilidad de utilizar un exponente de 10 que le permite convertir un valor en su equivalente en milisegundos. Por ejemplo, para convertir el valor desde de segundos utilice 3, para milisegundos -3. Cuando el valor ya se encuentre en milisegundos utilice 0. Nota: el inicio de los valores es según el estándar de tiempo UNIX (el 0 corresponde al 1ero de enero de 1970 a media noche).

Multiestado

De forma predefinida su representación será Plana, mostrando los valores inalterados del punto.

Para un punto Multiestado el generador le permite definir un número arbitrario de textos numéricos o de etiquetas para representar los valores de cada estado y sus colores. En el caso que un valor ocurra y no esté dentro de las etiquetas asignadas, se mostrará el valor crudo. Un conjunto típico de valores incluyen encendido/apagado/desactivado, o apagado/caliente/frío.

Alfanuméricos

De forma predefinida, y su única representación será Plana, mostrando los valores inalterados del punto y un opcional Sufijo.
