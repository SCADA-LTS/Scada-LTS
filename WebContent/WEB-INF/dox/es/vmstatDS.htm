Descripción General

La fuente de datos VMStat es utilizada para monitorear aspectos de rendimiento de servidores tipo *nix. Cuando la fuente de datos se inicia esta lanza un proceso que ejecuta el programa vmstat, y su salida es monitoreada.

Configuración

Cada fuente de datos requiere un Nombre, que puede ser cualquier descripción. La configuración de Segundos entre encuesta determina cuán frecuentemente el proceso vmstat reportará de regreso la información al Scada. La configuración de Fuera de escala puede ajustarse automáticamente ajustada en diferentes escalas de valores medidos en byte: bytes, k, KB, m, y MB.
