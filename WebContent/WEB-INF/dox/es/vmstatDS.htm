Visión general

La fuente de datos del VMStat se utiliza para monitorear varios aspectos del desempeño de un huésped *nix. Cuando se inicia la fuente de datos, se inicia un proceso que ejecuta el programa vmstat, y se monitoriza la salida de este proceso.

Configuración

Cada fuente de datos requiere un Nombre, que puede ser cualquier descripción. El parámetro Segundos de encuesta determina la frecuencia con la que el proceso vmstat reportará los datos a Scada-LTS. El parámetro Escala de salida puede ser ajustado automáticamente para ajustar los valores de los bytes a diferentes escalas, incluyendo bytes, k, KB, m y MB.
