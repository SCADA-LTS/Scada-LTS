Visión general

La fuente de datos del SNMP se utiliza para recoger datos de los dispositivos que implementan el protocolo SNMP. Esta fuente de datos realiza consultas y también acepta trampas SNMP.

Configuración

Cada fuente de datos requiere un Nombre, que puede ser cualquier descripción. El período Actualización determina la frecuencia con la que los puntos SNMP serán consultados para obtener datos. Los otros campos son necesarios para configurar los detalles de la comunicación SNMP.

Prueba de anfitrión NSP

La prueba de anfitrión puede utilizarse para comprobar tanto la configuración de la fuente de datos como la existencia y el valor de un determinado OID (Identificador de objetos). Ingrese el OID que se va a probar y haga clic en "Obtener el valor del OID". La configuración actual se utilizará para enviar una solicitud de SNMP. Si tiene éxito, se mostrará el valor OID. De lo contrario, se proporcionará un mensaje que describa el error.
