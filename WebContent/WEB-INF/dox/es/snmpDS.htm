Descripción General

La fuente de datos SNMP se utiliza para recopilar datos desde un dispositivo que tenga una implementación del protocolo SNMP. Esta fuente de datos realiza encuestas SNMP y también acepta Trap.

Configuración

Cada fuente de datos requiere un Nombre, que puede ser una descripción. El Período de actualización determina cuán frecuentemente los puntos SNMP serán encuestados para obtener información de ellos. Los campos restantes son necesarios para configurar correctamente la comunicación SNMP.

Prueba del Servidor SNMP

la prueba del servidor puede utilizarse para verificar la correcta configuración de la fuente de datos y la existencia de un objeto específico OID (Identificador de Objeto). Introduzca el OID a verificar y haga clic en "Obtener valor OID". Los valores de configuración se utilizarán para enviar una solicitud GET del SNMP (get request). Al ser exitosa la solicitud, se mostrará el valor correspondiente al OID. Cuando falla la solicitud, es desplegado un mensaje con el detalle del error.
