El parámetro HTTP nombre es la clave utilizada para entregar los datos a la fuente de datos del receptor HTTP. Se proporciona como la parte 'pointName' en valores de puntos formateados como 'pointName=pointValue' o '__pointFoo=pointName&__valueFoo=pointValue'. Puede utilizar la función de escucha del receptor HTTP en la configuración de la fuente de datos del receptor para determinar los nombres de los parámetros disponibles en sus solicitudes.

El Tipo de datos es cualquier tipo de datos válido. Véase el resumen del sistema para obtener más información sobre los tipos de datos.

Si el tipo de datos es Binario, se debe proporcionar un valor Binario 0. Si el valor del punto de la petición coincide con este valor, el valor asignado al punto será 0; de lo contrario será 1.
