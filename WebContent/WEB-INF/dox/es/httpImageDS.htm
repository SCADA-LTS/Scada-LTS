Visión general

Esta fuente de datos se utiliza para reunir imágenes de sitios web disponibles en la Internet, o de equipo interno disponible en una intranet. Utilizando mecanismos similares a los de un navegador web, la fuente de datos solicita información a los dispositivos habilitados para HTTP y luego almacena las imágenes en un área protegida del sistema de archivos. Esta es una fuente de datos de encuestas.

Configuración

Cada fuente de datos requiere un Nombre, que puede ser cualquier descripción. El Período de Actualización determina la frecuencia de la petición HTTP.
