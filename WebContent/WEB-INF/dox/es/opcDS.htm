Descripción

La fuente de datos OPC permite leer los datos de un servidor OPC

.

Configuración

Todas las fuentes de datos requieren un Nombre, que puede tener cualquier descripción.

El anfitrión indica la dirección local del cliente.

El Dominio indica el dominio donde se encuentra el cliente. PD: El cliente debe estar en un dominio para trabajar con OPC.

El Usuario indica el nombre del usuario del sistema operativo - OS.

La Contraseña indica la contraseña del sistema operativo.

El Servidor indica qué servidores están disponibles para los datos proporcionados.

El Período de actualización indica el período de actualización de los datos de la fuente de datos.

El Modo de creación indica cómo se crearán las etiquetas OPC en el cliente. Tenemos dos opciones: Examinar Etiquetas - donde se mostrarán todas las Etiquetas disponibles en el Servidor y Agregar Etiquetas - donde se agregará una Etiqueta disponible del Servidor.
