Visión general

Esta fuente de datos se utiliza para reunir información genérica de los correos electrónicos recibidos en un determinado buzón POP3. Los mensajes en el buzón pueden haber sido recibidos de cualquier fuente. Utilizando mecanismos similares a los de un cliente de correo, la fuente de datos se conecta al buzón POP3 y procesa los mensajes en él individualmente. Los mensajes se borran después de ser procesados. Se utilizan expresiones regulares para extraer la información deseada. Esta es una fuente de datos de encuestas.

Configuración

Cada fuente de datos requiere un Nombre, que puede ser cualquier descripción. El Período de verificación del buzón determina la frecuencia con la que se revisa el buzón en busca de mensajes. Servidor POP3 es el nombre de dominio o dirección IP del servidor que aloja el buzón. Para la conexión se utiliza el puerto estándar POP3 110. El nombre de usuario del buzón y el contraseña del buzón deben ser proporcionados para el acceso.

El área Test Content es donde se puede probar una muestra de contenido de correo electrónico para comprobar el RegEx de un punto. Para más información, vea la documentación sobre "Propiedades de los puntos POP3".
