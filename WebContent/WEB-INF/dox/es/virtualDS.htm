La fuente de datos virtual es una fuente de datos sin ninguna conexión con un sistema externo o físico. Existe completamente dentro del espacio de memoria del sistema. Los datos que produce son aleatorios o deterministas, dependiendo de la configuración de los puntos.

Las fuentes de datos virtuales pueden ser útiles de varias maneras:

  • Una forma sencilla de validar que su sistema funciona
  • Puede crear datos predecibles con los que se pueden comparar los datos reales.
  • Puedes controlar el equipo de una manera más compleja. Por ejemplo, si la iluminación de su casa está controlada por el sistema, puede utilizar un punto binario aleatorio para encender y apagar las luces mientras está de vacaciones, haciendo que parezca que siempre está en casa.
  • Cualquier lugar donde los datos deterministas o aleatorios sean útiles

La configuración de la fuente de datos es simple. Sólo requiere un nombre y el período de tiempo entre cada actualización de los datos. Por ejemplo, si elige "2 minutos", entonces cada dos minutos se ejecutarán los algoritmos que cambian los valores de los puntos virtuales para esta fuente de datos.
