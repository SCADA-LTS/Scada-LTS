Descripción General

Una fuente de datos Virtuales una fuente de datos sin conexión a sistemas externos o físicos. Este tipo de fuente de datos sólo existe en el espacio de la memoria del servidor. Los datos provienen de generaciones aleatorias, o determinísticas, dependiendo de como configure los puntos.

Una fuente de datos virtual puede ser útil en casos como:

  • Es la vía más simple de validar que el sistema está funcionando.
  • Puede generar datos predictivos con los cuales hacer comparaciones de funcionamiento con sus datos reales.
  • Puede controlar equipos de formas complejas. Por ejemplo, si las luces de una una casa son controladas por el sistema, puede crear puntos binarios aleatorios para encender y apagar las luces cuando los dueños estén de vacaciones, creando así la impresión que están un en casa.
  • Cualquier aplicación donde los datos aleatorios o determinísticos sean de utilidad.

La configuración de la fuente de datos es simple, sólo req1uiere de un nombre, y del intervalo de tiempo entre cada actualización. Por ejemplo, si elige "2 minutos", entonces cada dos minutos se cambiarán los valores del punto virtual según su algoritmo.
