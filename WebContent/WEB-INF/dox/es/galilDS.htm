Visión general

La fuente de datos de Galil se utiliza para intercambiar datos con el equipo Galil DMC-21x2 accesible a través de una red I/P. El equipo puede estar ubicado en una red de área local o en una intranet, o también puede estar ubicado en cualquier lugar de la Internet. Es una fuente de datos de encuestas.

Configuración

Cada fuente de datos requiere un Nombre, que puede ser cualquier descripción. Los parámetros Anfitrión y Puerto determinan cómo encontrar el equipo en la red. El host puede ser un nombre de dominio o una dirección IP.Los campos Tiempo de espera y Reintentos determinan cómo se comportará el sistema si la consulta falla. La fuente de datos espera el número de milisegundos de tiempo de espera dado para una respuesta del equipo. Si no se recibe, la solicitud se volverá a intentar el número de veces especificado. El período Actualización determina la frecuencia con la que el equipo es interrogado para obtener datos.

Una vez que se establece una conexión con el equipo, se mantiene viva durante todo el tiempo que el equipo lo permita. Las conexiones cerradas causan un intento de reapertura automática.

Ordene la prueba

La conectividad y el estado del equipo pueden ser probados usando la sección Test del comando Galil. Esta función abre una nueva conexión a la que enviar el comando, de modo que no entre en conflicto con ninguna fuente de datos en ejecución. Sin embargo, si el dispositivo sólo permite un número limitado de conexiones simultáneas, la conectividad puede fallar.

El valor Comando puede ser cualquier cadena de comandos válida (excluyendo el terminador CRLF, y posiblemente excluyendo un punto y coma de cierre). El resultado del comando se mostrará bajo el botón Ejecutar. Si la conexión con el equipo ha fallado, se mostrará el mensaje de error. Si no se recibe respuesta, la conexión se terminará después del tiempo de demora dado. Si se recibe una respuesta de error (es decir, si se recibe '?'), se mostrará el mensaje "(respuesta de error)".
