Configuración

Se puede añadir un solo punto, definiendo su índice, o Añadir múltiples puntos de datos definiendo los índices mínimo y máximo.

Ordenes

Para los puntos del tipo OUTPUT, debe definir al comando los parámetros que se utilizarán como predeterminados, si no se especifican durante el tiempo de ejecución. Para los puntos de tipo SALIDA ANUAL, los comandos válidos (que deben ser ejecutados en la lista de vigilancia, así como otras fuentes de datos) son:

pon/poff -> pulso on/off

lon/loff -> cierre encendido/apagado

cerrar/viaje -> cerrar/viaje

Todos los comandos pueden ser seguidos por dos parámetros, el comando TIEMPO ENCENDIDO y TIEMPO APAGADO en milisegundos. Por ejemplo: pon,500,200

Sólo se puede introducir "1", "0", "verdadero" o "falso". En este caso, el pedido enviado será el pedido por defecto especificado en el punto.

Ejemplo

Punto binario registrado como CERRAR/VIAJE, TIEMPO DE VIGILANCIA 1000 y TIEMPO LIBRE 0

1 -> el sistema enviará el comando CERRAR con TIEMPO DE VIGILANCIA 1000 y TIEMPO LIBRE 0

0 -> el sistema enviará el comando TRIP con TIEMPO DE VIGILANCIA 1000 y TIEMPO LIBRE 0

pon -> el sistema enviará el comando PULSE ON con TIEMPO DE VIGILANCIA 1000 y TIEMPO LIBRE 0

poff,300,0 -> el sistema enviará el comando PULSE OFF con TIEMPO DE VIGILANCIA 300 y TIEMPO LIBRE 0
