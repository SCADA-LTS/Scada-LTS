Las listas de correo pueden ser agregadas para manejar múltiples direcciones de correo de forma simple y simultánea. En ellas se pueden referenciar direcciones de correo electrónico de usuarios tanto en servidores externos como internos. Estas listas pueden ser utilizadas como destinatarios de los envíos de correos de los manejadores de eventos.

Utilice el icono para crear una nueva lista de correos,o edite una lista existente haciendo clic en su nombre. De esta forma también podrá enviar correos a este nombre desde otras áreas del sistema.

Cuando se edita una lista, utilice el icono para agregar un usuario o para especificar una dirección externa al sistema. Tenga presente que los cambios al editarle a un usuario del sistema su dirección de correo electrónico requiere hacer una actualización en las listas que este esté definido. Una entrada en de un usuario del sistema se identificará con un icono de usuario (el icono depende del tipo de usuario); y una dirección de correo explicita se mostrará con el icono . Para eliminar una entrada en la lista haga clic en el icono junto a la dirección del usuario a remover.

Para guardar los cambios en una lista, haga clic en el icono . Para eliminar toda una lista utilice el icono . Para enviar un correo de prueba a todas las entradas de la lista, haga clic en el icono .
