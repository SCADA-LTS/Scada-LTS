Los formatos Fecha/hora se utilizan para determinar cómo mostrar e interpretar las fechas, convertir sus representaciones internas y el texto mostrado. Las fechas pueden presentarse en diferentes formatos, según el contexto y la semántica de los datos. Los mismos estándares pueden utilizarse para mostrar un valor o introducir un valor.

En Scada-LTS el formato está implementado por la clase Java SimpleDateFormat, puedes obtener los detalles de esta clase en el enlace "http://java.sun.com/j2se/1.4.2/docs/api/java/text/SimpleDateFormat.html".

El formato puede depender de la ubicación predeterminada de su Java VM.
