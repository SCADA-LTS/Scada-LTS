El Dispositivo remoto IP es la dirección IP en la que se puede encontrar el dispositivo. El puerto del dispositivo remoto es el número de puerto en el que escucha las solicitudes.

El Tipo de objeto es el tipo de objeto para el que se solicitarán valores. El Número de instancia del objeto es el identificador único del objeto/tipo cuyos valores se solicitarán. Los puntos solicitan el Valor actual de una instancia de objeto.

El Utilizando la suscripción de VOC determina si Mango intentará suscribirse a los eventos de VOC para el objeto en el dispositivo. No todos los objetos admiten suscripciones de COV, y no todos los dispositivos admiten suscripciones de COV, incluso para los objetos que normalmente lo harían.

Utilice Establecimiento para determinar si la interfaz permitirá a los usuarios introducir el valor con el que actualizar el objeto. Sólo los valores de "salida" se consideran configurables.

El campo Tipo de datos determina el tipo de datos de Mango que tendrá el punto. Los valores devueltos de la red BACnet se convertirán automáticamente al tipo requerido.
