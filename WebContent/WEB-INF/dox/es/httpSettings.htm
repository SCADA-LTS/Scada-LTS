Este conjunto de parámetros de configuración permiten determinar cómo se realizarán las solicitudes de HTTP (HTTP requests) desde la ubicación de ScadaBR. Estas solicitudes serán utilizadas por la fuente de datos del receptor HTTP (HTTP Retriever), el Publicador HTTP Publisher (HTTP Publisher), y otros mecanismos internos del sistema (incluyendo la obtención de esta documentación).

Su administrador de Red podrá suministrarle si su red utiliza un proxy, y de ser así, cual configuración utilizar.
