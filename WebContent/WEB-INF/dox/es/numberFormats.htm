Los Formatos Numéricos son utilizados para determinar como se mostrarán y analizarán los valores numéricos, así se convierten de números de punto flotante manejados internamente por enteros o sus representaciones en texto. Por ejemplo, el número 1234.5678 puede ser formateado como se muestra, pero puede preferirse mostrarlo como '1,234.6', o '1 234,567', o '1.2345678E3' según el contexto y la semántica del valor. Los mismos patrones pueden ser utilizados para dar formato a un valor a visualizar, y para analizar un texto de entrada para un valores numéricos.

El formato numérico en Mango está implementado en la clase DecimalFormat de Java, los detalles de esta clase se pueden obtener en aquí. Algunos patrones comúnmente utilizados:

  • #,##0.00 Separador de miles utilizando coma. Siempre se mostrará el primer dígito entero y dos lugares decimales (redondeados).
  • 0.000 Siempre mostrar el primer dígito entero y tres lugares decimales.
  • 0.00;(0.00) Siempre mostrar el primer dígito entero y dos lugares decimales. Los números negativos se representan entre paréntesis.

Los formatos pueden ser localizados según sus valores iniciales de la Java VM.
