Formatos de números se utilizan para determinar cómo mostrar y analizar los valores numéricos, convirtiéndolos entre representaciones internas de punto flotante o enteros y texto. Por ejemplo, el número 1234.5678 puede formatearse como se muestra, pero es mejor verlo como "1.234.6", o "1.234.567", o "1.2345678E3" dependiendo del contexto y la semántica del valor. Las mismas plantillas se pueden utilizar para dar formato a un valor para su visualización, así como para analizar el texto introducido en valores numéricos.

El formato de los números en Mango se implementa mediante la clase de formato decimal de Java, cuyos detalles se pueden encontrar aquí. A continuación se proporcionan algunas plantillas comunes para su conveniencia.

  • #,##0.00 Separa las comas en miles. Siempre muestra el primer número entero y dos decimales (redondeados).
  • 0.000 Siempre muestra el primer número entero y tres decimales.
  • 0.00 ;(0.00) Siempre muestra el primer número entero y dos decimales. Muestra los números negativos entre paréntesis.

El formato puede ser localizado de acuerdo con la configuración predeterminada de su Java VM.
