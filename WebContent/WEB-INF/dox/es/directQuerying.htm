Scada-LTS ofrece la posibilidad de acceso directo a la base de datos del sistema, tanto para consultarla como para actualizarla. Esta función está destinada principalmente a la solución de problemas o a la reparación del sistema. No es adecuado para su uso regular debido al riesgo de corrupción de datos.

Sólo se pueden enviar declaraciones SQL individuales válidas. No termines tus declaraciones con caracteres ";".
