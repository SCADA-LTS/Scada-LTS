El ScadaBR tiene la capacidad de acceder directamente a la Base de Datos del Sistema, tanto para consultas, como para actualizaciones. Esta capacidad está diseñada; principalmente para solucionar problemas del sistema o repararlo. Utilizar esta capacidad para su uso regular puede ocasionar corrupción del sistema.

Sólo utilice expresiones válidas de SQL. No finalice las declaraciones con el carácter ';'.
