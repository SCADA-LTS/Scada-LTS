Visión general

Una referencia a un punto JMX consiste en un Nombre del objeto, un Nombre del atributo, y opcionalmente - si es un tipo de datos compuestos - un nombre del elemento compuesto. La forma más fácil de encontrar los valores es usar la función de inspección JMX del servidor.

El punto Tipo de datos es la representación Scada-LTS del valor, no el tipo de valor JMX. De cualquier manera, es mejor que el tipo de datos coincida con el tipo JMX para obtener mejores resultados.
