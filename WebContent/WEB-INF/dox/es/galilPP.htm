Visión general

La mayoría de los tipos de puntos en una fuente de datos de Galil reflejan los valores en el equipo de Galil. La excepción es el tipo de punto de control, que es más abstracto. A continuación se dan detalles.

Todos los tipos de puntos requieren un nombre para la pantalla, que puede ser cualquier valor.

Ordenes

El tipo de punto de mando permite enviar comandos arbitrarios al equipo de Galil. Los comandos pueden ser introducidos en la lista de monitoreo, desde vistas gráficas o a través de manejadores de eventos de punto de ajuste. El valor del comando y el valor de la respuesta se registrará en el historial de puntos (uno tras otro), a menos que se produzca un error, en cuyo caso se registrará el comando y se desencadenará un evento que describa el error.

Entrada

Los tipos de punto de entrada usan los comandos @IN y @AN para recuperar los valores actuales. Estos puntos no son configurables. El Tipo de datos determina qué comando se usará para consultar el valor. El Número de entrada es el índice del valor a recuperar. Para los puntos numéricos, la escalada lineal del valor también se puede hacer de forma automática ajustando los valores Escala en consecuencia.

Salida

Los tipos de puntos de salida usan los comandos CB, SB y @OUT para recuperar y establecer valores de salida binarios. El número de salida Salida es el índice del valor a manejar.

Indique la posición

Los tipos de puntos de posición Tell utilizan el comando TP para recuperar la posición de un eje en particular. Estos puntos no son configurables. El Axis es la etiqueta del eje requerido. El escalado de los valores lineales puede hacerse automáticamente estableciendo los valores Escala en consecuencia. Seleccione Redondear a entero para redondear los valores decimales del valor de recuperación (y escalado).

Variable

Los tipos de punto variable utilizan el comando "Var=" para definir y recuperar los valores de las variables en el equipo. El comando Nombre de la variable es el nombre de la variable deseada. Los valores que se definan y reciban se convierten automáticamente según el Tipo de datos dado.
