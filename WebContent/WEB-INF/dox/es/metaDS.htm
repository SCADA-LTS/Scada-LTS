Descripción General

Una fuente de datos Meta toma su nombre porque permite combinar puntos existentes en nuevos puntos. En lugar de tomar la información de una fuente de datos externa, utiliza los valores de puntos internos, y los manipula a gusto.

JavaScript / ECMAScript

Este tipo de fuente de datos requiere conocimiento de JavaScript, y de ECMAScript. (Las especificaciones completas de ECMAScript se encuentran en aquí.) Para muchos, JavaScript es el lenguaje más popular para la programación de scripts de hoy en día, es el único lenguaje disponible para todos los navegadores populares. Como resultado esta popularidad es el uso común del JavaScript en las páginas webs. El contexto dentro de las fuente de datos Meta es diferente, no hay objetos "ventana" o "documentos", además de otras diferencias. Junto a las diferencias contextuales, una implementación completa de ECMAScript está disponible en las fuentes de datos. Específicamente la implementación Rhino es utilizada.
