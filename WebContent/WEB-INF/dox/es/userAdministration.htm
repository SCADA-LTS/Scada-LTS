Visión general

Los usuarios son las personas que usan el sistema. Se pueden asignar diferentes privilegios a los usuarios para permitirles leer, modificar los datos o gestionar la configuración del sistema.

Hay dos niveles de usuarios: administradores (), y otros usuarios (). Los administradores tienen acceso a todas las características del sistema. Otros tienen acceso limitado dependiendo de su configuración.

Edición de usuario

Para añadir un usuario, haz clic en el icono . El usuario no será añadido hasta que se haya registrado con éxito. Para editar un usuario, haga clic en el nombre del usuario en la lista de "Usuarios".

Las contraseñas no pueden ser descifradas una vez guardadas; sólo pueden ser restablecidas con una nueva contraseña.

Un usuario discapacitado no puede conectarse. Para borrar un usuario, haz clic en el icono .

Para probar la dirección de correo electrónico de un usuario, haga clic en el .Se enviará un mensaje a la dirección especificada.

Los usuarios pueden recibir correos electrónicos aunque no estén en la lista de correo de un administrador de eventos. Si se selecciona el nivel de alarma Enviar alarma por correo "ninguno" se asegurará que estas notificaciones se envíen cuando el nivel de alarma del evento sea mayor o igual que el nivel de alarma mínimo del usuario, pero sólo cuando se active el evento (es decir, no se produzcan escaladas o notificaciones inactivas). Tenga en cuenta que si el usuario ya está presente en el Administrador de Eventos, no recibirá correo duplicado.

Derechos del usuario

La casilla de verificación Administrador determina si el usuario es un administrador.

Si el usuario no es un administrador, se mostrarán todas las fuentes de datos disponibles. Si se marca la casilla junto al nombre de la fuente de datos, el usuario podrá modificar los ajustes de la fuente de datos y acceder a todos los datos producidos por la misma.

Si no se verifica la fuente de datos, se mostrará una lista de todos los puntos de la fuente de datos. Cada punto puede ser ajustado individualmente para lectura, escritura o, si el punto es ajustable, acceso de lectura. (Tener acceso de escritura implica tener acceso de lectura).
