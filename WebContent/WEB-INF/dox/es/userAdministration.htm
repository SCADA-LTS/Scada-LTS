Resumen

Los Usuarios representan individuos que utilizan el sistema. Diferentes tipos de privilegios les pueden ser asignados para garantizarles la habilidad de leer, modificar y leer datos, además de modificar el comportamiento del sistema.

Hay dos niveles básicos para los Usuarios: los Administradores (), y el resto de los usuarios (). Los Administradores tienen acceso a todas las funcionalidades del sistema. Lo demás usuarios tienen privilegios que se basan en la configuración que se les realice.

Editando los Usuarios

Para agregar un nuevo usuario haga clic en el icono . El usuario será adicionado únicamente al ser guardado. Si desea editar un usuario existente, selecciónelo de la lista de usuarios haciendo clic en su nombre.

Las contraseñas no pueden ser recuperadas luego que son asignadas; sólo podrá reiniciar una contraseña especificando una nueva.

Un usuario deshabilitado no podrá ingresar al sistema. Para eliminar un usuario haga clic en el icono .

Para probar una dirección de correo haga clic en el icono . Esto enviará un correo a la dirección indicada.

Los usuarios pueden recibir en sus correos notificaciones de alarmas sin necesidad de realizar una configuración específica en el manejador de eventos. Al seleccionar un nivel de alarmas en Enviar correo de alarmas se enviará el nivel correspondiente y sus inferiores, y al seleccionar "Ninguno" no será enviado ningún correo para este usuario por este concepto.

Note que si ya se ha especificado el correo del usuario en un evento del Manejador de Eventos, el sistema sólo enviará un correo aún que esté configurado en varias opciones similares (no se le enviará un duplicado del mismo correo).

Privilegios de los Usuarios

La casilla del Administrador determina si este usuario será un administrador.

Cuando el usuario no es administrador se listarán todas las fuentes de datos y sus puntos. Marcando la casilla correspondiente al nombre de la fuente de datos permitirá que el usuario pueda alterar sus configuraciones, de igual forma accederá a todo lo relacionado con esta fuente de datos.

Cuando está desmarcada la casilla de la fuente de datos, se mostrará la lista de puntos configurados en ella. El acceso a cada punto será individualmente configurado para: Sin acceso, lectura, o escritura (cuando el punto sea modificable).

Nota: El acceso de escritura a un punto implica que el usuario podrá leer este punto.
