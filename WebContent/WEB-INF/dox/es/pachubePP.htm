El Tipo de datos es la representación del valor de Mango. Los datos recibidos se convierten automáticamente en el tipo de datos requerido.

El Feed ID y el Data stream ID se utilizan para identificar de forma única un flujo de datos. Las solicitudes de múltiples corrientes dentro de una sola corriente se consolidan para mayor eficiencia.

Para el tipo de datos binarios, se puede establecer el valor Binario 0. Si el valor encontrado en el contenido coincide, el punto se actualiza con 0, de lo contrario el punto se actualiza con 1.

Para asegurarse de que su flujo de datos existe, haga clic en el icono asociado .
