Descripción General

El tipo de dato es la representación de un valor en Mango. Al recibir un dato, este es automáticamente convertido en el tipo de dato requerido.

El Id de Feed y el Id del flujo de datos son utilizados unicamente para identificar un flujo de datos. Las solicitudes consolidadas en un flujo único, cuando provienen de múltiples flujos, mejoran la eficiencia.

Para los tipos de datos binarios, puede definir el valor binario 0. Cuando se encuentra un valor que coincida con este, el punto será actualizado a cero (0). De otra forma se actualizará con el valor de 1.

Para asegurar que el flujo de datos existe, haga clic en el icono asociado .
