Utilice la configuración de Idioma del Sistema para seleccionar el idioma que el sistema utilizará cuando se encuentre una situación con idioma desconocido. Por ejemplo, informes y mensajes de correo que no están asociados con un usuario específico o con varios. Para este caso será tomado como idioma de comunicación el configurado para el sistema.

NOTA: El ScadaBR utiliza gran variedad de herramientas que no están cubiertas en el i18n, por lo que puede recibir mensajes extraños en inglés.

Generando una nueva traducción

El ScadaBR es un sistema con completa capacidad multilenguaje. Ahora, si el ScadaBR no contiene un idioma determinado, este puede ser fácilmente agregado con una simple traducción de archivos. De esta forma tendrá la instancia del "idioma" que requiere.

Para adicionar un idioma inicie con la edición del archivo "WEB-INF/classes/i18n.properties" añadiendole la información del idioma correspondiente. El formato de su contenido corresponde a "<código del idioma>=<nombre del idioma>", donde "código del idioma" es tomado del estándar de código ISO 639-1 para idiomas, y el "nombre del idioma" es el nombre como se le conoce en su propio idioma. Esta nueva línea agregará el lenguaje a la lista de idiomas seleccionables del sistema.

Ahora, debe realizar la traducción de la lista mensajes del sistema. Cree un nuevo archivo en el directorio "WEB-INF/classes/" y nómbrelo "messages_<código del idioma>.properties". Para simplificar el proceso es útil copiar el contenido del archivo de mensajes en Inglés, y hacer las traducciones correspondientes.

Por último, se debe traducir la documentación contextual (p.e: este documento de ayuda). Para esto cree un nuevo directorio en "WEB-INF/dox/" con el nombre del código del lenguaje, y realice una versión traducida de todos los documentos contenidos en el directorio de Idioma Inglés ("en").

Para garantizar que los archivos traducidos permanecerán en las nuevas versiones de ScadaBR, por favor continúe contribuyendo con Serotonin Software donde lo fusionarán en el repositorio principal.
