Resumen

Esta funcionalidad sirve para que pueda exportar su actual proyecto en el ScadaBR. Se generará un archivo .zip con todos los archivos necesarios para una posterior recuperación del proyecto.

Nombre y Descripción del Proyecto
El nombre y la descripción del proyecto permiten identificar un proyecto mientras se está importando.
Incluir el histórico de puntos
Los datos almacenados en la base de datos por el histórico de los puntos pueden ocupar mucho espacio en disco. Seleccione esta opción únicamente si realmente requiere exportar toda la información de la historia de los puntos junto con el resto del proyecto. La exportación y la importación, de los puntos históricos puede llevar un tiempo considerable.
Incluir la carpeta de "Uploads"
Esta contiene los datos que fueron enviados por el sistema, Por ejemplo: las imágenes de fondo para las representaciones gráficas.
Incluir la carpeta de Gráficos
La carpeta de gráficos (Graphics) es la carpeta que contiene la información de las representaciones gráficas. Inclúyala cuando ha agregado manualmente nuevos componentes gráficos al sistema.
