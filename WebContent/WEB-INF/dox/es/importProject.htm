Resumen

Esta funcionalidad permite realizar la importación de un proyecto completo (.zip) hacia el ScadaBR. Atención: TODOS los datos del actual proyecto serán eliminados!

Importando
Seleccione el archivo .zip del proyecto dentro de su computadora y haga clic en el botón Enviar. El sistema tomará el archivo y mostrará las informaciones básicas del Proyecto y su Descripción. Verifique que seleccionó el proyecto correcto y comience con la importación (ésta puede demorar unos minutos).
Cuidados
* Luego de iniciar una importación, espere que esta finalice.
* Recuerde que TODOS los datos del proyecto actual serán eliminados.
* Si el proyecto fue exportado incluyendo el histórico de valores de puntos, el proceso podrá durar varios minutos.
* Los Usuarios declarados en el sistema serán eliminados y los contenidos en el proyecto serán insertados. Asegúrese que conoce al menos un usuario del nuevo proyecto para poder acceder al sistema.
