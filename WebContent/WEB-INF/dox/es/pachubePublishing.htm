Descripción General

El publicador Pachube envía cada actualización del punto (o su cambio) a Pachube utilizando la clave API ingresada, el Id del feed, y el ID del flujo de datos como solicitudes separadas de HTTP.

Propiedades

La Clave API es necesaria para actualizar datos confidenciales. En el área de "mi configuración" se define cuando iniciará sesión en su cuenta Pachube. Los parámetros tiempo de espera y reintentos son utilizados para manejar fallas.

respuesta HTTP (HTTP response)

Cualquier código de respuesta diferente a "200 OK" es considerado como un error, y se levantará una alarma. Note que - dependiendo de su cuenta en Pachube - pueden aplicar algunas restricciones en las solicitudes API. Vea http://community.pachube.com/api#rate_limit para más detalles.

Puntos Publicados

La sección de Puntos lista los puntos a ser publicados. Un punto publicado puede ser habilitado () o deshabilitado (). El deshabilitar un punto permite que no sea utilizado sin necesidad de eliminarlo. Los valores de Id de Feed y de Id de flujo determina el flujo de datos que será actualizado.

Para agregar un punto a la lista de puntos, selecciónelo en la lista de puntos y haga clic en el icono Para eliminar un punto existente utilice el icono a su lado.
