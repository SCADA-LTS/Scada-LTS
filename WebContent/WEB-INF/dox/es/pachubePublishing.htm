Visión general

El editor de Pachube envía cada actualización (o cambio, según sea el caso) a Pachube usando la clave API, el ID de flujo y el ID de flujo de datos dado como una solicitud HTTP separada.

Propriédad

La API Key es necesaria para actualizar los datos privados. Lo encontrarás en el área de "mi configuración" cuando entres en tu cuenta de Pachube. Los ajustes Tiempo muerto y Reintento se usan para manejar las fallas.

Respuestas del HTTP

Cualquier código de respuesta que no sea "200 OK" se considera un error, y se disparará una alarma. Ten en cuenta que, dependiendo de tu cuenta de Pachube, pueden aplicarse restricciones a las solicitudes de la API. Ver http://community.pachube.com/api#rate_limit para más detalles.

Artículos publicados

La sección Puntos enumera los puntos que se publicarán. Los puntos de publicación pueden ser habilitados () o deshabilitados (). Desactivar un punto permite no usarlo sin tener que borrarlo explícitamente. Los valores ID de la alimentación y ID del flujo de datos determinan el flujo de datos que se actualizará.

Para añadir un punto a la lista de puntos, selecciónalo en la lista de puntos y haz clic en el icono . Para eliminar un punto existente, haga clic en el icono asociado a él.
