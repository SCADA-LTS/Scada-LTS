Este panel permite la modificación de varios atributos de visualización para el punto de vista. Si se introduce, el valor Sustitución del nombre del punto se mostrará en la ventana emergente de información del punto en lugar del nombre dado al punto. Si no se introduce, se muestra el primer nombre. El valor Settable override hace que un punto ajustable aparezca como no ajustable en el contexto de la vista.

El Color de fondo reemplaza el color por defecto de la interfaz. Si se deja en blanco, se utiliza el color por defecto. Introduzca "transparente" para no mostrar un color de fondo o de borde (útil si la imagen de fondo de la vista ya proporciona un contexto de fondo para el punto de vista). De lo contrario, se puede introducir cualquier valor aceptable como color CSS. Sólo el color valor debe ser proporcionado. Por ejemplo, los valores válidos para el rojo incluyen (excluyendo las comillas) "#FF0000", "rojo", "#F00", "rgb(255, 0, 0), y "rgb(100%, 0%, 0%)". Una simple descripción de los valores de color válidos se puede encontrar aquí. Para detalles completos, ver el W3C.

La casilla Mostrar controles determina si los controles de vuelo se mostrarán para un punto de vista. Eliminar los controles de los puntos de apoyo puede reducir el desorden en las vistas densas. También puede dar una visión más "estática", lo que puede ser útil para vistas remotas.
