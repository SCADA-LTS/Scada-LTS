Este panel permite la modificación de los atributos de despliegue. Este panel permite la modificación de los atributos de las diferentes visualizaciones para las vistas de un punto. Si se introduce, nombre del punto a reemplazar el valor de reemplazo mostrará en la información del punto emergente en lugar del nombre del punto. Si no se introduce, el nombre original es mostrado. La configuración de reemplazo permite al punto modificable mostrarse como no modificable en el contexto de la vista.

El color de fondo para los reemplazos, inicialmente es el color de la interface. Si dejamos este valor en blanco, se utilizará el color predefinido. Ingrese en la casilla el valor 'transparent' (sin las comillas) para prescindir del color del fondo y de los bordes para ese punto (esto es útil cuando existe una imagen de fondo con estos contextos para el punto). Por otro lado, cualquier valor es aceptado bajo el código CSS. Sólo debe introducir el valor del color deseado. Por ejemplo, los valores válidos para el rojo incluyen (sin utilizar las comillas) "#FF0000", "red", "#F00", "rgb(255, 0, 0), y "rgb(100%, 0%, 0%)". Puede encontrar una descripción simple para los colores válidos en aquí. Para información más detallada, vea W3C

La casilla de controles de vista determina si estos se muestran para el punto en el gráfico. deshabilitar estos controles de los puntos puede disminuir la complejidad en vistas densas. Esto es útil cuando desea obtener una vista más "estática", particularmente aplicable para vistas remotas.
