El rendimiento de la interfaz de usuario representa la capacidad de respuesta de la interfaz de usuario. Esto incluye todas las actividades que son gestionadas por el mecanismo de sondeo largo, como el indicador de nivel máximo de alarma, la actualización de la lista de vigilancia, la actualización de la lista de eventos, etc. Si el host de Mango tiene una CPU de potencia limitada, se puede configurar el rendimiento de la interfaz de usuario a un nivel inferior para evitar que los navegadores de los usuarios consuman demasiada CPU del host.

Si se marca la casilla "Enviar errores a la serotonina", las excepciones que esta instancia de Mango observe serán enviadas a la serotonina para su revisión. Si está modificando la fuente de Mango para sus propios fines, por favor, ponga este valor en desmarcado.

Utilice la configuración de "eventos de purga" para establecer la edad máxima de un evento antes de que sea purgado de la base de datos. La purga borra permanentemente el evento y cualquier comentario del usuario asociado a él.

Utilice la configuración de "Purgar instancias de informes" para definir la edad máxima de una instancia de informe antes de ser purgada de la base de datos. La purga elimina permanentemente la instancia de informe. Es preferible fijar este valor lo más bajo posible porque los informes pueden consumir potencialmente mucho espacio de almacenamiento disponible. La purga puede evitarse estableciendo el atributo "prevenir la purga" en la instancia del informe.

Haga clic en el botón Purgar todos los datos para borrar permanentemente todas las muestras puntuales que se encuentran actualmente en la base de datos. Esto puede ser útil para dar un "nuevo comienzo" a una instancia de Mango, especialmente si existen datos erróneos en las actividades de configuración y de puesta en marcha.

Utilice los ajustes de "Rechazar valores de puntos en una fecha posterior" para definir el tiempo máximo en el futuro en que los puntos de datos aceptarán nuevos valores. Cualquier valor con una marca de tiempo que exceda este valor será rechazado, y se escribirá un mensaje de error en el registro.
