Visión general

Esta fuente de datos se utiliza para reunir información genérica de Pachube. Los datos se organizan en Pachube en flujos, que son colecciones de flujos de datos. Dado que un flujo de datos corresponde a un punto de Mango, la definición de un punto de Pachube se hace especificando tanto un identificador de flujo como de corriente. Es una fuente de datos de encuestas.

Configuración

Cada fuente de datos requiere un Nombre, que puede ser cualquier descripción. El Período de Actualización determina la frecuencia de la petición HTTP a Pachube. Ten en cuenta que, dependiendo de tu cuenta de Pachube, pueden aplicarse restricciones a las solicitudes de la API. Ver http://community.pachube.com/api#rate_limit para más detalles.

La tecla API es necesaria para acceder a los datos privados. Lo encontrarás en el área de "mi configuración" cuando entres en tu cuenta de Pachube. Los campos Tiempo de espera y Reintenta determinan cuánto tiempo hay que esperar a que el servidor web devuelva los datos, y cuántas veces hay que reintentar la conexión en caso de fallo. Si se agotan todos los intentos, se desencadena un evento de excepción de la fuente de datos.
