Все числовые объекты Вашего текущего списка, которые позволяют строить графики/таблицы, могут быть объединены в общий график/таблицу при помощи раздела Графики/таблицы. Чтобы вывести график необходимо нажать на пиктограмму Запуск (). Графики не обновляются автоматически, для того чтобы не перегружать систему. Для обновления графика вручную необходимо нажать на пиктограмму Запуск
