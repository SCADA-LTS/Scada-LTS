Обзор

Источник метаданных называется так потому, что он может комбинировать существующие объекты в новые объекты. Вместо того, чтобы извлекать данные из внешнего источника, он использует значения других объектов, позволяя им перемещаться произвольным образом.

JavaScript / ECMAScript

Это источник данных требует знания Java-сценариев, также известных как ECMA-скрипты. (Полную спецификацию по ECMA-скриптам можно найти здесь). Java-скрипт является, вероятно, наиболее популярным языком из используемых в настоящее время, и единственным языком, поддерживаемым во всех популярных веб-браузерах. С этим связано его широкое применение в веб-браузерах. Контекст внутри источника метаданных, конечно, отличается. В нем нет объектов типа «окно» или «документ», есть и другие менее значимые отличия. Если отложить в сторону контекстные отличия, в данном источнике можно пользоваться всеми преимуществами ECMA-скрипта. Например, можно воспользоваться реализацией Rhino.
