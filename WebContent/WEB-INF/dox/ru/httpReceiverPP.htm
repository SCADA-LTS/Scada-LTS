Имя параметра HTTP – это ключ, используемый для доставки данных в приемник HTTP. Он выглядит как блок 'pointName' в данных объекта, сформатированных либо в виде выражения 'pointName=pointValue' либо '__pointFoo=pointName&__valueFoo=pointValue'. Чтобы получить имена параметров, используемых в Ваших запросах, Вы можете использовать функцию прослушивания HTTP-приемника в конфигурации источника данных.

Тип данных – этой любой действительный тип данных. Более подробная информация о типах данных приводится в разделе «Общий обзор системы».

Если тип данных двоичный, то нужно указать Двоичное значение 0 (Binary 0 value). Если значение объекта в запросе соответствует этому значению, то объекту присваивается значение 0, в противном случае – 1.
