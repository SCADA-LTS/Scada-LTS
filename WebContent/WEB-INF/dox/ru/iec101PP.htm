Конфигурация

На сегодняшний день поддерживается 3 типа данных: Одноэлементный объект,
Двухэлементный объект и Нормализованное значение измеренной величины (Single Point, Double Point and Normalized Measure).

Команды

Чтобы отправлять команды, вы должны указать Смещение (Offset), которое будет добавляться к адресу объекта при отправке команды. Например, при записи адреса объекта 2 со смещением 2000, команда будет отправлена объекту 2002.
