Конфигурация

Каждый источник данных должен иметь одно произвольно заданное Имя.

Адрес источника является адресом главного устройства или адресом источника данных, который необходимо зарегистрировать. Адрес подчиненного устройства – это адрес удаленного устройства, с которым необходимо установить связь.

Значение поля Период опроса RBE соответствует интервалу времени, по истечении которого система отправляет подчиненному устройству требование (запрос) классов 1, 2 и 3 (события). Постоянный период опроса соответствует интервалу времени, по истечении которого система направляет устройству требование (запрос) класса 0 и который имеет следующее отношение к периоду опроса RBE: если Период опроса RBE = 2 с, а постоянный период опроса = 30, то источник данных будет направлять устройству запросы класса 0 каждые 60 секунд.
