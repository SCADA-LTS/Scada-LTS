Обзор

Этот источник данных используется для сбора обобщенной информации из электронных сообщений, полученных в заданные почтовые ящики по POP3-протоколу. Сообщения могут поступать в почтовые ящики из любого источника. При помощи механизмов, аналогичных механизмам клиента электронной почты, источник данных устанавливает соединение с почтовым ящиком по протоколу POP3 и индивидуально обрабатывает каждое сообщение, найденное в нем. После обработки сообщения удаляются. Для выборки требуемой информации используются регулярные выражения. Этот источник данных работает в режиме опроса.

Конфигурация

Каждый источник данных должен иметь произвольно заданное Имя. Значение поля Период проверки входящей почты задает частоту проверки почтового ящика на наличие входящих сообщений. Сервер POP3 – это имя домена или IP-адрес сервера, на котором размещается почтовый ящик. Для соединения используется стандартный порт 110. Для входа требуется ввести Имя пользователя входящей почты и Пароль входящей почты.

В поле Проверить содержимое можно ввести тестовое электронное сообщение, чтобы проверить регулярное выражение объекта. Более подробная информация приводится в разделе «Свойства объекта POP3».
