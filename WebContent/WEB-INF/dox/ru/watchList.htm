Обзор

Список наблюдения – эта табличное представление объектов, которых вы хотите просматривать одновременно. Значения объектов и время значений обновляются автоматически, т.е. нет необходимости обновлять страницу. (На самом деле, система работает даже более эффективно, если вы не выполняете ручное обновление).

Именованные листы наблюдения

Пользователь может вести несколько списков наблюдения, но просматривать одновременно можно только один список. Чтобы создать новый список наблюдения, нажмите пиктограмму в правом верхнем углу раздела списков наблюдения. По умолчанию списку присваивается имя "(unnamed)". Чтобы его переименовать, наведите курсор мыши на пиктограмму . Появится панель редактирования имени. Введите новое имя и нажмите Enter или кнопку «Сохранить», чтобы сохранить имя. Чтобы удалить список наблюдения, нажмите пиктограмму в верхнем правом углу раздела списков наблюдения. Каждый пользователь должен иметь. по крайней мере, один список наблюдения, поэтому пиктограмма удаления активна только в случае, если списков наблюдения более одного.

Точки в листах наблюдения

Точка может принадлежать нескольким листам (спискам) наблюдения, поэтому вы можете создавать любые необходимые Вам листы.

Данные, которые отображаются для каждой точки в Вашем списке, зависят от конфигурации этой точки. Точки, которые являются устанавливаемыми (и для настройки которых у Вас есть соответствующие права) будут показаны пиктограммой (). Если навести курсором мыши на эту пиктограмму, появится окно с возможными элементами настройки данного объекта. Точки, которые сконфигурированы под графики, имеют пиктограмму графика (). Если навести курсором мыши на эту пиктограмму, на экран выведется текущий график. Графики обновляются автоматически, также как значения и их время, поэтому при появлении новых данных график сам автоматически обновится. Пиктограмма параметров объекта () переместит Вас на страницу параметров выбранной точки.

Чтобы поменять сортировку точек в Вашем списке, используйте значки перемещения. Нажатие на значок переместит объект в списке на одну строку ниже, а нажатие на значок переместит его на одну строку выше.
Чтобы добавить объекты в текущий список, нажмите на имя объекта в разделе Объекты. Он будет вставлен в конец списка. Чтобы удалить объекты, нажмите пиктограмму удаления () справа от записи объекта. Нажатие на эту пиктограмму удалит объект из Вашего текущего списка наблюдения, но не удалит его из системы. Не следует путать пиктограмму Удалить объект с пиктограммой Удалить список наблюдения в верхней части страницы.

Подсветка

Значения в списке наблюдения и их время кратковременно подсвечиваются при их изменении. Подсветка остается до завершения следующего цикла обновления. Значения времени обычно подсвечиваются больше, чем сами значения, так как при обновлении объекта время всегда изменяется, а значение объекта – не всегда.
