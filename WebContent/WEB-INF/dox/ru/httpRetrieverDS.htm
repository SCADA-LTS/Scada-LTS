Обзор

Этот источник данных используется для сбора обобщенной информации с веб-сайтов, доступных в сети Интернет или с внутреннего оборудования, подключенного к Интранет-сети. Используя механизмы работы аналогичные веб-браузеру, источник данных запрашивает информацию с поддерживающих протокол HTTP устройств, а затем использует регулярные выражения для вычленения требуемых данных. Этот источник данных работает в режиме опроса.

Конфигурация

Каждый источник данных должен иметь произвольно заданное Имя. Период обновления определяет частоту HTTP-запросов. URL – это адрес страницы, которую необходимо обработать. Для тестирования URL нажмите пиктограмму . (Откроется новое окно или закладка браузера).

Значения полей Время ожидания и Повторы определяют время, которое будет затрачено на ожидание загрузки страницы с веб-сервера, и количество повторных попыток в случае ошибки соединения. Если количество попыток исчерпано, генерируется событие об исключительной ошибке источника данных.
