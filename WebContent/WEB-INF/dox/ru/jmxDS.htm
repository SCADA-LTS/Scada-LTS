Обзор

Источник данных JMX используется для отслеживания значений, выдаваемых сервером JMX, либо локальным, либо удаленным. Имеющиеся значения очень сильно отличаются друг от друга в зависимости от приложений, работающих в JVM. Но всегда есть общие значения, например, значения, связанные с памятью JVM.

Конфигурация

Каждый источник данных должен иметь произвольно заданное Имя. В окошке метки Использовать локальный JMX-сервер задается требование использовать JMX-сервер в локальной JVM. Если флажок не проставлен, то необходимо указать Адрес удаленного JMX-сервера.

Период обновления определяет частоту опросов JMX-сервера. Если поставить флажок Отсчет, то запуск источника данных будет происходить с задержкой, равной «округленному» значению времени. (например, если период опроса равен 10 секундам, то запросы будут отправляться на 10-ой секунде, затем на 20-ой секунде и т.д.). В противном случае опрос будет начинаться без задержки.
