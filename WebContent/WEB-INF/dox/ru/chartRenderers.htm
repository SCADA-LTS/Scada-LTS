Обзор

Построитель графиков/таблиц используется для объединения и вывода на экран данных о состоянии объекта в течение определенного периода времени или для некоторого количества его значений. Совокупность данных объекта представляется в зависимости от типа данных значений. По существу, в списке выбора Типа данных предлагаются только те типы, которые соответствуют текущему типу данных объекта.

Внимание! Построители графиков/таблиц основываются на данных, записанных в журнале. Если вы задали для данного объекта тип регистрации «Не архивировать данные», скорее всего, вы не получите нужного результата на экране. Исключением являются таблицы, в которых используются значения объекта, записанные в кэш-памяти, но только в тех случаях, когда эти значения есть. Данные, записанные в кэш-памяти, сбрасываются при перезапуске системы.

Статистические данные

Этот построитель отображает таблицу статистической информации, рассчитанной на основе значений объекта в течение указанного Периода времени. Выводимая информация зависит от типа данных объекта. Статистическая информация доступна для всех типов данных.

Двоичное представление и представление в нескольких состояниях: при помощи этого типа данных формируется статистика «запусков и исполнений» для каждого состояния объекта в течение указанного периода времени. Каждая запись содержит количество изменений состояния («запусков») и общее количество времени, в течение которого объект был в режиме исполнения, а также процентное соотношение времени исполнения к полному периоду времени.

Числовое представление: объекты с числовым типом данных дают следующие статистические значения в течение указанного периода времени: минимальное значение и время его фиксации, максимальное значение и время его фиксации, среднее значение, сумма (применяется для подсчета импульсов).

Алфавитно-цифровое представление: количество изменений состояния объекта в заданный период времени.

Графики

Этот построитель выдает на экран линейный график (гистограмму), построенный по значениям объекта в указанный период времени. Эта функция доступна для объектов с числовым типом данных или с типом данных с несколькими состояниями.
