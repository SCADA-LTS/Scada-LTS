В системе есть возможность установить конкретное значение уровня сигнала тревоги. Для того, чтобы задать уровень тревоги какого-либо системного события, используйте меню выбора уровня тревоги, связанное с этим событием. Возникающее событие будет иметь присвоенный ему уровень тревоги только после сохранения значений этого параметра. Сохраните Ваши настройки, нажав на пиктограмму текущего раздела.
