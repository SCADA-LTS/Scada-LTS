Overview

Источник данных Spinwave используется для сбора данных с устройств Spinwave, подключенных к локальному шлюзу и доступных через последовательные интерфейсы RS232 или RS485 (требуется преобразователь). Этот источник данных получает регулярные обновления с подключенных устройств; он не выполняет опрос.

Конфигурация

Каждый источник данных должен иметь произвольно заданное Имя. Для выбора Порта предлагается список всех доступных последовательных портов в оборудовании хоста. Необходимо задать порт, к которому подключен шлюз Spinwave. Значение Версии сообщения зависит от устройства, которое вы используете. Более подробную информацию вы найдете в документации на оборудование Spinwave.

Поиск Адресов сенсоров обычно выполняется при помощи функции прослушивания сенсоров, описанной ниже. Однако, если вы знаете уникальные адреса Ваших устройств, вы можете ввести их вручную. После ввода адреса, нажмите пиктограмму и добавьте его в список известных адресов.

Программа прослушивания сенсоров Spinwave

Как только шлюз Spinwave подключается к хосту, можно запустить программу, которая будет прослушивать поступающие сообщения. Обычно устройства настроены на отправку сообщений каждую 21 секунду, но эту настройку можно изменить. Чтобы добавить в Список адресов сенсоров адреса всех прослушанных устройств, нажмите кнопку «Добавить найденные сенсоры в список».
