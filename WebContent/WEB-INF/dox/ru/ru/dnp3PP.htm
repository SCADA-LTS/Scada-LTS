Конфигурация

Вы можете добавить один отдельный объект, указав его индекс, или вы можете Добавить группу объектов данных, определив минимальный и максимальный индексы.

Команды

Для объектов типа OUTPUT (ВЫВОД) Вам необходимо определить параметры команд, которые будут использоваться по умолчанию, в случае, если они не должны задаваться в режиме исполнения. Для объектов типа BINARY OUTPUT (ДВОИЧНЫЙ ВЫВОД) существуют следующие команды (которые должны исполняться при нажатии соответствующей пиктограммы листа наблюдения, а также других источников данных):

pon/poff -> импульс вкл/выкл
lon/loff -> триггер-защелка вкл/выкл
close/trip -> закрыто/ход

Все команды сопровождаются двумя параметрами: время запуска команды и время остановки команды в миллисекундах. Например, pon,500,200
Существует возможность ввести только "1", "0", "true" или "false". В этом случае отправленная команда будет считаться командой по умолчанию, назначенной для этой точки.

Пример

Двоичная точка типа ЗАКРЫТО/ХОД, ВРЕМЯ ЗАПУСКА 1000 e, ВРЕМЯ ОСТАНОВКИ 0

1 -> система отправит команду ЗАКРЫТЬ со временем запуска 1000 и временем остановки 0.

0 -> система отправит команду ХОД со временем запуска 1000 и временем остановки 0

pon -> система отправит команду ИМПУЛЬС ВКЛ. со ВРЕМЕНЕМ ЗАПУСКА 1000 и ВРЕМЕНЕМ ОСТАНОВКИ 0

poff,300,0 -> система отправит команду ИМПУЛЬС ВЫКЛ. со ВРЕМЕНЕМ ЗАПУСКА 300 и ВРЕМЕНЕМ ОСТАНОВКИ 0.
