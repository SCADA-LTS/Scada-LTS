IP удаленного устройства - это IP-адрес, по которому можно найти устройство. Порт удаленного устройства – это номер порта, который прослушивает запросы.

Тип объекта – это тип объекта, которому отправляются запросы на получение значений. Номер экземпляра объекта – это уникальный идентификатор объекта/типа, с которого запрашиваются значения. Объекты запрашивают текущее значение экземпляра объекта.

Значение поля Использовать подписку на получение COV определяет, будет ли Mango пытаться установить подписку на COV-события объекта устройства. Не все объекты поддерживают подписку на получение COV, и не все устройства поддерживают подписку, даже если объект этого устройства обычно ее поддерживает.

Для того, чтобы дать пользователю права на ввод значений для обновления объекта, используйте критерий Настраиваемый объект. Настраиваемыми могут быть только выходные значения.

Поле Тип данных определяет, какой тип данных Mango будет иметь значение объекта. Значения, полученные из сети BACnet, будут автоматически преобразованы в требуемый тип.
