На данной странице приводятся элементы управления для редактирования свойств объекта. Пиктограмма осуществляет переход на страницу свойств объекта в режиме «только чтение»; она доступна только для ранее сохраненных объектов.

Необходимо задать Имя объекта, это может быть любое текстовое значение. Имя будет использоваться для визуальной идентификации объекта в системе.

Имя источника данных объекта приводится для справки. Пиктограмма является ссылкой на страницу редактирования источника данных, где можно найти элементы конфигурации локатора значений. Эта функция используется для удобства.
