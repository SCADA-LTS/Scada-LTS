Списки наблюдения можно использовать совместно с другими пользователями. Эти пользователи не имеют прав на редактирование списка наблюдения (хотя они могут удалить себя из (отписаться от) совместно используемого списка). Чтобы использовать совместно с другими пользователями список наблюдения, который вы создали, выберите пользователя из выпадающего окна и нажмите на пиктограмму . Затем выберите тип доступа, которым будет обладать пользователь, либо «чтение» (без права установки значений) либо «установка» (с правом установки значений). Вы можете добавить любое необходимое количество пользователей. Изменения в настройках совместного использования сохраняются автоматически. Чтобы удалить пользователя из совместно используемого списка, нажмите на пиктограмму , связанную с записью этого пользователя.
