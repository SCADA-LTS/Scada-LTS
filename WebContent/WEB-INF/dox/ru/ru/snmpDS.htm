Обзор

Источник данных SNMP используется для сбора данных с устройства, которое реализует протокол SNMP. Этот источник данных выполняет опрос, а также принимает SNMP-ловушки.

Конфигурация

Каждый источник данных должен иметь произвольно заданное Имя. Значение поля Период обновления определяет частоту опроса объектов SNMP. Остальные поля нужны для редактирования настроек связи SNMP.

Проверка хоста SNMP

Проверка хоста может быть использована для проверки конфигурации источника данных, а также проверки наличия и значения заданного Идентификатора объекта (OID). Введите OID, который необходимо протестировать, и нажмите кнопку «Получить значение OID». Текущие настройки конфигурации будут использования для отправки SNMP-запроса. При успешном выполнении запроса, на экране выводится значение OID. В противном случае, выводится сообщение с описанием ошибки.
