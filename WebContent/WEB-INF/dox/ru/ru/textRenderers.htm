Обзор

Назначением построителя (визуализатора) текста является преобразование необработанных значений в форму, более понятную и информативную для пользователя. Способы визуализации необработанных значений зависят от типа данных значения. Поэтому, при Выборе типа визуализатора выдается список только подходящих типов данных текущей точки.

Двоичные данные

Двоичный визуализатор позволяет пользователю определить текстовую метку, которая представляет каждое из двух состояний (т.е. 0 или 1). Можно задать любую метку. Кроме того, для каждого состояния можно задать цвет. Примерами таких меток могут быть: Выкл./Вкл., стоп/пуск, ложь/истина и т.п., но, при необходимости, можно задать любой другой текст.

Простой визуализатор – это двоичный визуализатор, используемый по умолчанию. Он отображает неизмененное необработанное значение – 0 или 1

Числовые данные

Аналоговый визуализатор позволяет пользователю задавать Формат значения, в котором он может определить такие параметры, как количество знаков после запятой, следует ли разделять тысячи и т.п. (Смотри документацию «Числовые форматы»). Для отображения единиц измерения можно добавить значение из поля Суффикс. Суффиксы могут содержать HTML-объекты, такие как ° (записывается '&deg;') для вывода на HTML страницы. Более подробную информацию по HTML-объектам смотрите на http://www.w3.org/TR/REC-html40/sgml/entities.html

Простой визуализатор – это числовой визуализатор, используемый по умолчанию. Он отображает неизмененное необработанное значение.

Визуализатор диапазона позволяет пользователю задавать Формат числа (Смотри раздел «Числовые форматы»), а также произвольное количество текстовых значений для представления различных числовых диапазонов от/до. Каждому из этих диапазонов также может быть назначен определенный цвет. Значения в числовом формате выдаются только в том случае, если они не попадают в заданный диапазон. Например, для объекта измерения температуры могут быть заданы диапазоны «Холодный», «Нормальный» и «Горячий» с соответствующим цветом для каждого.

Визуализатор времени позволяет пользователю задавать Формат даты/времени (более подробную информацию смотри в разделе «Формат даты/времени»). Обычный формат даты – «МММ Д, ГГГГ». Обычный формат времени - «ЧЧ:ММ:СС». Форматы даты и времени при необходимости можно комбинировать. Также для преобразования значения в миллисекунды можно задать экспоненту значения времени. Это показатель степени числа 10, который будет использоваться для преобразования заданного значения в эквивалентное значение в миллисекундах. Например, чтобы преобразовать значение из секунд в миллисекунды, задайте 3, из микросекунд в миллисекунды – задайте -3. Если значение уже указано в миллисекундах, задайте 0. В системе принято, что значения времени в миллисекундах начинают отсчет в полночь, 1 января 1970 г.
Данные с несколькими состояниями
Простой визуализатор – это визуализатор значений с несколькими состояниями, используемый по умолчанию. Он выводит на экран неизмененное необработанное значение.

Данные с несколькими состояниями

Простой визуализатор – это визуализатор значений с несколькими состояниями, используемый по умолчанию. Он выводит на экран неизмененное необработанное значение.

Визуализатор значений с несколькими состояниями дает возможность пользователю задать произвольное количество текстовых значений для представления различных состояний объекта. Каждому из этих состояний также можно назначить определенный цвет. Если возникает состояние, которому не назначена метка, отображается необработанное значение. Обычный набор меток содержит: Вкл./Выкл./Неактивно или Выкл./Тепл./Хол.

Алфавитно-цифровые данные

Простой визуализатор - это только алфавитно-цифровой визуализатор. Отображает неизмененное необработанное значение, включая необязательный суффикс.
