Тип данных – это тип представления значений в Mango. Полученные данные автоматически преобразовываются в требуемый тип.

Идентификатор ленты и идентификатор потока данных используется для однозначного определения потока данных. Запросы на группу потоков одной ленты консолидируются для повышения эффективности.

Для двоичного типа данных можно установить Двоичное значение 0. Если значение, найденное в содержимом, совпадет с установленным, объект будет обновлен со значением 0, в противном случае, он будет обновлен со значением 1.
Чтобы убедится в существовании конкретного потока данных, нажмите связанную с ним пиктограмму .
