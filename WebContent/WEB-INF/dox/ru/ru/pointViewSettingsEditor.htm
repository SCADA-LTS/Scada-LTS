Данная панель позволяет изменять различные атрибуты вида объекта. При задании параметра Замена имени точки во всплывающем информационном меню точки, вместо заданного имени точки будет отображаться значение. Если параметр не задавать, отображаться будет имя точки. Установка функции Замена устанавливаемой точки позволяет отображать устанавливаемую точку как обычную в контексте вида.

Поле Цвет фона задает цвет фона вида взамен установленного по умолчанию цвета. Если оставить поле незаполненным, используется цвет по умолчанию. Введите значение «прозрачный», чтобы фона не было или на экране отображалась граница вида (это удобно, если фон вида уже задан фоновым рисунком). В остальных случаях можно задать любое значение цвета из CSS. В поле должно указываться только значение цвета. Например, правильные значения для красного цвета (без кавычек): "#FF0000", "red", "#F00", "rgb(255, 0, 0), и "rgb(100%, 0%, 0%)". Краткое описание значений цветов можно найти здесь. Исчерпывающую информацию можно найти на W3C.

Флажок Управляющие элементы экрана задает возможность отображения элементов управления, появляющихся при наведении курсора мыши на компоненты вида объекта. Чтобы уменьшить загруженность экрана в насыщенных компонентами видах, такие управляющие элементы можно отключить. В этом случае вид выглядят более «статично», это особенно удобно использовать для удаленных видов.
