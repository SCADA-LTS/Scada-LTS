Обзор

Источник данных сети 1-wire используется для сбора данных из локальной сети 1-wire, доступной через последовательные интерфейсы RS232 или RS485 (необходим преобразователь). Это источник данных, работающий в режиме опроса.

Физическая конфигурация сети может произвольно меняться в процессе её работы, не создавая помех эксплуатации и работоспособности источника данных. При обнаружении изменений в топологии сети автоматически выполняется повторное сканирование. Если положение объектов после предыдущего сканирования не изменилось, событие не возникает

Конфигурация

Каждый источник данных должен иметь произвольно выбранное Имя. При выборе Порта предлагается список всех доступных последовательных портов, имеющихся в аппаратном обеспечении хоста. Необходимо выбрать порт, к которому подключается сеть 1-wire. Значение поля Период обновления определяет частоту опроса сети.

Сканирование сети

Сеть можно сканировать при помощи утилиты сканирования. Эта функция позволяет выполнять сканирование сети по каналам связи, соединенным разветвителями, с целью поиска всех устройств, подключенных в сеть. Результаты поиска отражаются в таблице, где указаны адреса и описания устройств сети 1-wire. Чтобы быстро добавить найденное устройство в качестве объекта, необходимо нажать связанную с ним пиктограмму . Новый объект будет создан с атрибутами устройства по умолчанию.
