Описание

Источник данных OPC позволяет считывать данные с OPC-сервера.

Конфигурация

Каждый источник данных должен иметь произвольно заданное Имя.

В поле Хост задается локальный адрес клиента.

В поле Домен указывается расположение клиента. Примечание: чтобы работать с OPC, клиент обязательно должен быть в домене.

В поле Пользователь указывается имя пользователя из операционной системы – ОС.

В поле Пароль указывается пароль для входа в операционную систему.

Поле Сервер показывает доступные серверы для предоставленных данных.

В поле Период обновления задается значение периода обновления данных из источника данных.

Поле Режим создания задает способ создания OPC-тэгов клиента. Есть два варианта: Просмотр тэгов - выдается список всех доступных на сервере тэгов; и Добавление тэгов – с сервера добавляется доступный тэг
