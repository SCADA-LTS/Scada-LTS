Настройки Email используются для всех действий, связанных с отправкой и приемом электронных сообщений в данной версии программы.

SMTP host – это имя хоста, к которому обращается Ваш почтовый агент. Если Ваш хост требует авторизации, поставьте флажок в окошке Использовать авторизацию и заполните поля Имя пользователя и Пароль. Поставьте флажок в окне Включить TLS, если это нужно для Вашего сервера.

Адрес отправителя – это адрес, который будет использоваться в качестве адреса отправителя каждого сообщения. Адрес отправителя является полем, необязательным для заполнения. В нем можно указать псевдоним или собственное имя.

Чтобы выбрать тип содержимого, нужного Вам в электронных сообщениях Mango, используйте окошко выбора Типа содержимого. По умолчанию выставляется HTML и текст, в этих случаях Mango создает составное (multipart) MIME-сообщение, состоящее из частей "текст/html" и "текст/текст". Если выбрать тип Только HTML, сообщение также может быть составным, если в нем есть приложения или встроенное содержимое, но в остальных случаях оно будет состоять из одной части. Если выбран тип Только текст, сообщение всегда будет состоять из одной части. Примечание: запланированные сообщения-отчеты всегда отправляются в виде Только HTML из-за их содержимого.

После выполнения настроек, вы можете проверить их, нажав на пиктограмму . При этом на Ваш электронный адрес будет выслано тестовое сообщение со всеми настройками, которые вы произвели.
