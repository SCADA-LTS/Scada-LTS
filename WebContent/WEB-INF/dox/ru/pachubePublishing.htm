Обзор

Публикатор Pachube отправляет обновление (или изменение) каждого объекта в Pachube как отдельный HTTP-запрос с использованием заданных API-ключа, идентификатора ленты и идентификатора потока.

Свойства

API-ключ нужен для обновления частных данных. Вы можете найти его в разделе «мои настройки» при входе в учетную запись Pachube. Параметры Время ожидания и Повторы используются в случае возникновения ошибок.

HTTP-ответ

Любой код ответа, кроме «200 ОК», считается ошибкой и служит основанием для возникновения аварийного события. Обратите внимание, что в зависимости от Вашей учетной записи в Pachube, могут налагаться ограничения на API-запросы. Смотрите подробное описание по ссылке:
http://community.pachube.com/api#rate_limit

Публикуемые точки

В разделе Точки приводится список точке, которые должны быть опубликованы. Публикация точки может быть включена () или выключена (). Отключение точки позволяет не использовать его, при это не удаляя его полостью. Значения полей Идентификатор ленты и Идентификатор потока данных определяют поток данных, который будет обновляться.

Чтобы добавить точку в список объектов, выберите его в списке и нажмите пиктограмму (). Чтобы удалить существующую точку, нажмите связанную с ним пиктограмму ().
