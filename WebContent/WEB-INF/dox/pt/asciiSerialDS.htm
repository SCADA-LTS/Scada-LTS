Descrição

Este datasource serve para a leitura de dados oriundos da porta serial. O sistema aplicará as regex definidas pelo usuário no arquivo e colocará seus valores nos datapoints correspondentes.

Configuração

Todo datasource exige um Nome, que pode ter qualquer descrição.

A Porta deve indicar qual é a porta serial que está sendo usada para conexão.

O Baud Rate indica a taxa de transmissão.

Os Bits de Dados indica quantos bits serão transmitidos por vez.

Os Bits de Parada indica quantos bits de parada vão existir.

A Paridade indica se existirá ou não paridade.

O Período de Atualização indica o intervalo de tempo em a porta serial escutará por novos dados.

As Tentativas indica quantos tentativas de conexão serial realizadas.

O Timeout (ms) indica quanto tempo cada tentativa de conexão irá aguardar para obter sucesso.

O Modo de parada indica o modo de parada dos dados oriundos da portal serial. São possíveis três escolhas: Número de caracteres, Até o caracter C e por Timeout.

O Número de Caracteres indica quantos caracteres serão recebidos da porta serial por vez.

O Critério por caracter indica se a escolha será por caracter ASCII ou Hexadecimal.

O Caracter ASCII indica qual caracter ASCII será o critério de parada.

O Valor hexadecimal indica qual caracter Hexadecimal será o critério de parada.

O Timeout de parada (ms) indica que o critério de parada será por Timeout após a quantidade definida de milissegundos.

O Tamanho do Buffer indica o tamanho do buffer que os dados da porta serial ocuparão.

A String de inicialização indica qual String deve ser enviada pela porta serial para iniciar a conexão serial.
