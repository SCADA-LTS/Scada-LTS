Visão Geral

O data source SNMP é usado para obter dados de dispositivos que implementam o protocolo SNMP. Este data source executa pollings e também aceita SNMP traps.

Configuração

Cada data source requer um Name, que pode ser qualquer descrição. O Update period determina quão freqüente os pontos SNMP serão "pollados" por dados. O restante dos campos são necessários para configurar detalhes da comunicação SNMP.

Testando SNMP hosts

O teste de hosts pode ser usado para testar tanto a configuração do data source e a existência de determinado OID (Object IDentifier). Entre o OID a testar and clique "Get OID value". As configurações atuais serão utilizadas para enviar uma requisição SNMP get. Se a requisição for bem sucedida, o valor do OID será exibido. Senão uma mensagem descrevendo o erro será fornecida.
