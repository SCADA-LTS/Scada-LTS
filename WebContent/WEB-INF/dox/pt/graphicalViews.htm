Geral

Uma representação gráfica é semelhante a uma watch list, porém provê uma representação gráfica dos data points selecionados. Os valores dos data points e timestamps são atualizados automaticamente—você não precisa recarregar a página. (Na verdade, o sistema opera mais eficientemente se não recarregar)

Selecionado uma representação gráfica

Um usuário pode ter múltiplas representações gráficas, porém pode visualizar apenas uma por vez. Use o controle "dropdown" para selecionar a tela que você deseja. Para criar uma nova, clique no ícone () no canto direito superior da seção de watch list. Para editar uma representação existente, clique no ícone (). Em outros casos, a página de edição de representações gráficas será mostrada.

Você pode ver representações gráficas que você não criou em sua lista. Neste caso, elas foram compartilhadas com você pelas pessoas que as criaram. Você com certeza poderá visualizá-las por completo, mas pode ser que você não tenha permissão de escrita de valores através da parte gráfica se você não recebeu essa permissão. Se você deseja não ter mais uma representação gráfica na sua lista, use o ícone para excluir () para não compartilhar mais a representação.

Imagens de fundo

Representações gráficas podem ser opcionalmente apresentadas com imagens de fundo. Imagens de fundo podem ser configuradas na página de edição.

Data points em visões gráficas

Um data point pode pertencer a múltiplas representações, para que você possa criá-las tão específicas quanto precisar.

A representação de cada data point em sua representação depende da configuração dos mesmos. Data points podem ser renderizados simplesmente com seu valor corrente, ou um ícone arbitrário que pode ser usado para visualmente representar os valores dos "points". Quando o usuário passa o mouse sobre o data point, um campo de controles será mostrado (ao menos que eles estejam explicitamente não permitidos). Os controles mostrados também dependem da configuração dos data points. Navegar sobre o ícone de informação () irá mostrar os valores correntes dos data points e o último valor mudado. Data points que são setáveis (e os quais você tem permissão para setar) irão mostrar um ícone para escrita (). Navegando o mouse sobre esse ícone irá mostrar as opções para alterar o data point. Data points que são configurados para a parte gráfica irão mostrar um ícone de gráfico (). Navegando sobre esse ícone irá mostrar o gráfico atual. Os gráficos também são atualizados automaticamente, então quando um novo dado está disponível o gráfico irá atualizar automaticamente. Se um ícone de aviso () é mostrado do lado de um data point, isso indica que ou o data point ou seu data source estão desativados. Um indicador de espera () significa que o data point atualmente não tem valor, e está esperando por um.

Full Screen

No modo Full Screen, o conteúdo da Representação Gráfica apresentado na tela será somente os componentes presentes na Representação. Ou seja, o restante como o cabeçalho não será apresentado.

Para sair do modo Full Screen utilize o seguinte atalho: "Ctrl+Shift+F".
