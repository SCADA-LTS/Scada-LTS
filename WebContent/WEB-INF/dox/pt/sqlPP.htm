Os atributos utilizados para configurar pontos do data source SQL diferemThe attributes used to configure SQL data source points differ levemente dependendo de se a query é row-based ou column-based. Em ambos os casos, o Tipo de dado determina o tipo de dado do data point no ScadaBR. Valores do result set são automaticamente convertidos a esse tipo.

Para row-based queries, o Identificador de linha é comparado à primeira coluna do result set para encontrar a linha ou as linhas que contem valores (a segunda coluna) para este ponto. Tempos de sobreposição são tomados da terceira coluna (se existir).

Para column-based queries, o Nome da coluna corresponde ao nome da coluna no result set do qual o valor do ponto é extraído. A coluna Tempo de sobreposição (opcional) faz com que a coluna use tal tempo como o tempo do ponto. Observe que múltiplos pontos podem usar o mesmo tempo de sobreposição se for apropriado.

Instruções de atualização

Uma Instrução de atualização pode ser opcionalmente definida para um ponto, a qual determina como o valor do ponto pode ser configurado na base de dados. Trata-se de uma instrução padrão SQL usando "update" ou "insert". Se esta instrução não for definida, o ponto será considerado não-configurável.
