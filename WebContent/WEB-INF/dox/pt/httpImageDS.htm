Descrição

Este data source é usado para coletar imagens de web sites disponíveis na internet, ou equipamentos internos disponíveis em uma intranet. Usando mecanismos similares a de um navegador, o data source solicita informação de dispositivo habilitados em HTTPs e salva as imagens em uma região protegida no sistema de arquivos. Este é um data source do tipo polling.

Configuração

Todo data source necessita de um Nome, que pode ser qualquer descrição. O Período de atualização determina a freqüência com que a solicitação HTTP é feita.
