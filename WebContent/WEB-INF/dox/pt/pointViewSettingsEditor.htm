Este painel permite a modificação de diversos atributos de representação do ponto. Se definido, o valor do campo Sobrescrever nome do data point será mostrado na janela de popup do ponto ao invés do nome real do ponto. Se não definido, o nome real será mostrado. A opção Configurável permite que pontos configuráveis apareçam como não-configuráveis no contexto da representação.

O campo Cor de fundo sobrescreve a cor padrão da interface. Se deixado em branco, a cor padrão é utilizada. Entre com 'transparent' para não ter um fundo ou uma borda visível (útil se a imagem de fundo da representação gráfica já fornece um contexto de fundo para a representação do ponto). Caso contrário, qualquer valor aceitável como uma cor CSS pode ser definida. Seomente o valor da cor deve ser fornecido. Por exemplo, valores válidos para vermelho incluem (sem as aspas) "#FF0000", "red", "#F00", "rgb(255, 0, 0), e "rgb(100%, 0%, 0%)" Uma simples descrição de valores válidos de cor podem ser encontrados aqui. Para maiores detalhes veja a W3C

A opção Exibir controles determina se os controles flutuantes serão mostrados para uma representação de ponto. Remover os controles flutuantes de um ponto pode reduzir a bagunça em uma representação gráfica densa. Além disso, pode também dar uma sensação mais "estática" à represenatção, o que pode ser útil para representações remotas.
