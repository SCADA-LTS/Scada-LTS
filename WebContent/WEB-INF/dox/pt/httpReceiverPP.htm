O Nome do parâmetro HTTP é a chave usada para entregar os dados para o data source auditor HTTP. Ele é entregue como parte do 'nomeDoPonto' com valores do ponto formatados tanto como 'nomeDoPonto=valorDoPonto' ou como '__pointFoo=nomeDoPonto&__valueFoo=valorDoPonto'. Você pode usar a funcionalidade do auditor HTTP nas configurações do data source auditor HTTP para determinar quais os nomes de parâmetros estão disponíveis em sua requisição.

O Tipo de dado é qualquer tipo de dado válido. Veja a descrição do sistema para maiores informações sobre tipos de dados.

Se o tipo do dado for Binário, um Valor Binário 0 deve ser fornecido. Se o valor do ponto na requisição casar com este valor, o valor atribuído ao ponto será 0; caso contrário, será 1.
