A funcionalidade Descartar agora permite facilmente descartar valores de pontos a partir de uma base de tempo. Valores que são descartados por meio deste procedimento são permanentemente deletados e irrecuperáveis. Use os controles Descartar dados mais antigos que, ou selecione a opção Descartar todos as informações e clique no botão Descartar agora. O descarte é feito de modo assíncrono, assim você pode navegar para fora desta página ou alterar outras propriedades do ponto, mas fazendo isso você não receberá uma confirmação de descarte completo.
