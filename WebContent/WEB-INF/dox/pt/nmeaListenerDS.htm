Descrição

O data source auditor NMEA é utilizado para receber mensagens de equipamentos compatíveis com o protocolo NMEA 0183. Os dispositivos são conectados através de portas seriais RS232 e RS485 (são necessários os adaptadores). Este data source recebe atualizações regulares dos dispositivos conectados, mas não executa poll.

Configuração

Todo data source necessita de um Nome, que pode conter qualquer descrição. A seleção de Porta fornece uma lista de todos as portas seriais disponíveis no hardware que hospeda a aplicação. Isto corresponde à porta onde o equipmento NMEA está conectado. O valor de Baud rate é a taxa com a qual o equipamento pode comunicar suas mensagens. Este valor é tipicamente 4800, mas outros valores são possíveis, consulte a documentação do equipamento para detalhes. Valores de checksum são validados se são incluídos na mensagem. Se a validação do checksum falha, a mensagem é ignorada.

Auditor de mensagens NMEA

Assim que o equipamento NMEA estiver conectado ao hardware hospedeiro, um auditor pode ser iniciado para aguardar a chegada de mensagens. A taxa de chegada de mensagens depende do equipamento. As mensagens serão exibidas com os "envelopes" da mensagens excluídos, por exemplo, o indicador de início "$", o indicador de fim CRLF e dos valores "*" e checksum, caso estejam presentes. As mensagens exibidas são adequadas para copiar e colar em uma caixa teste de Regex de mensagem para teste nas regex dos pontos.
