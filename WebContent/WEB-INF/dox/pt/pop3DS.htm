Descrição

Este data source é utilizado para coletar informações genéricas de emails recebidos em uma dada caixa de email POP3. As mensagens recebidas na caixa de email podem ter sido recebidas por qualquer fonte. Usando mecanismos similares ao de um cliente de emails, o data source conecta à caixa de emails POP3 e individualmente processa as mensagens encontradas lá. As mensagens são deletadas após serem processadas. Expressões regulares são usadas para extrair as informações desejadas. Ele é um data source do tipo polling.

Configuração

Todo data source requer um Nome, o qual pode ser qualquer descrição. O Período de checagem de mensagens determina a frequência com que a caixa de emails é checada por mensagens. O Servidor POP3 é o nome do domínio ou o endereço IP do servidor que hospeda a caixa de emails. A porta padrão POP3 110 é utilizada para conexão. O Usuário e Senha devem ser fornecidos para acesso.

A área Testar conteúdo é onde um exemplo do conteúdo de um email pode ser posto para testar uma Expressão Regular. Veja a documentação "Propriedados de ponto POP3" para mais informações.
