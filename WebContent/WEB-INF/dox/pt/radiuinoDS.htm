Descrição

Este datasource serve para a leitura de dados oriundos de uma conexão serial com uma base Radiuino. O sistema irá buscar pelos sensores definidos nos datasources e coletar as informções de acordo com os bytes selecionados.

Configuração

Todo datasource exige um Nome, que pode ter qualquer descrição.

A Porta deve indicar qual é a porta serial que está sendo usada para conexão.

O Baud Rate indica a taxa de transmissão.

Os Bits de Dados indica quantos bits serão transmitidos por vez.

Os Bits de Parada indica quantos bits de parada vão existir.

A Paridade indica se existirá ou não paridade.

O Modo polling indica se o sistema deverá fazer o polling para coletar as informações ou se não irá fazer polling e aguardar que os sensores envie os dados periodicamente.

O Período de Atualização indica o intervalo de tempo que o sistema fará o polling por novos dados.

O Número de tentativas indica quantas tentativas de polling serão realizadas.

O Timeout (ms) indica quanto tempo cada tentativa de polling irá aguardar para obter sucesso.
