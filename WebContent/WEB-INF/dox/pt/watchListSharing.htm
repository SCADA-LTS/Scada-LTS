Watch lists podem ser compatilhadas com outros usuários. Elas não podem ser editadas pelo usuá com quem você você compartilha (mas esse usuário pode se retirar da lista de compartilhamento). Para compartilhar um watch list que você criou, selecione o usuári na caixa de seleção e clique no ícone . Então, escolha o tipo de acesso que este usuário deve ter, seja "read" (sem permissão para definir pontos) ou "set" (com permissão para definir pontos). Adicione tantos compartilhamentos à watch list quanto for necessário. Alterações às configurações de compartilhamento são salvas automaticamente. Para remover um usuário da lista de compartilhamento, clique no ícone associado ao usuário.
