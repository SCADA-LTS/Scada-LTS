Descrição

O publicador HTTP sender envia cada atualização de ponto (ou alteração, como for conveniente) para um dado URL de destino com uma requisição HTTP a parte.

Propriedades do HTTP sender

O Método determina qual método HTTP será usado para entregar os dados. Os dados enviados em ambos os casos são idênticos; a diferença é uma questão de estrutura de mensagem. O URL define o destino para o qual enviar os dados. Esquemas válidos incluem "http" e "https". O servidor pode ser um nome de domínio ou um endereço IP. Caminhos e strings de consulta também são válidos.

Uma lista de Parâmetros estáticos também podem ser adicionados. Estes parâmetros são incluídos em toda requisição enviada à URL. Desta maneira, este é um local apropriado para incluir credenciais de autorização ou outros artefatos de identificação. Para adicionar um parâmetro estático, digite sua chave e valor, e clique no ícone . Para deletar um parâmetro estático existente, clique no ícone associado com o parâmetro.

Resposta HTTP

Qualquer outro código de resposta que não seja um "200 OK" é considerado um erro, e um alarme será disparado. Além disso, qualquer resposta contendo uma resposta "OK" é considerada uma informação de aviso. A configuração Disparar aviso de resultado controla se a existência de qualquer resposta causa o disparo de um alarme de informação incluindo o conteúdo da resposta na mensagem do evento.

Testar configuração

Uma requisição de teste pode ser enviada — usando a cofiguração definida mas incluindo somente parâmetros estáticos — clicando no botão "Enviar parâmetros estáticos". O resultado da tentativa de envio será mostrado abaixo dos botões. Para cancelar um teste antes que ele termine, clique no botão "Cancelar".

Pontos publicados

A área Pontos lista os pontos a serem publicados. A publicação de pontos pode ser habilitada () ou desabilitada (). Desabilitar um ponto permite que ele não seja usado sem ter que explicitamente deletá-lo. O Nome do parâmetro é a chave que será usada para o ponto nos parâmetros da requisição. O valor será o valor presente do ponto. Selecione Incluir timestamp para incluir um parâmetro na requisição que contenha o tempo de atualização para o valor. A chave para o tempo será "__time", e a data será codificada usando o formato padrão "yyyMMddHHmmss". (veja a documentação "Formatação de Data/hora" para maiores informações.)

Para adicionar um ponto na lista de pontos, selecione-o na lista de pontos e clique no ícone . Para deletar um ponto existente, clique em seu ícone associado.
