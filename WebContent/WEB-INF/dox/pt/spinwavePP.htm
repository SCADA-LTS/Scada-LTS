A lista Endereços dos sensors corresponde à lista dos endereços conhecidos na configuração do data source. A seleção Atributos fornece uma lista de todos os atributos disponíveis para a versão de de mensagem selecionada.

Por padrão, valores de temperatuar são fornecidos em Fahrenheit. Para automaticamente converter valores para Celsius, selecione a opçãoe Converter para Celsius.
