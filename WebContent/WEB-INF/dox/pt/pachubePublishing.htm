Descrição

O publicador Pachube envia cada atualização de ponto (ou alteração, como for apropriado) para o Pachube usando a API key fornecida, o feed ID, e o data stream ID como uma requisição HTTP a parte.

Propriedades

A Chave da API é necessária para atualizar dados privados. Você pode encontrá-la na área "my settings" quando você entra em sua conta Pachube. Os parâmetros Timeout e Retries são usados para tratar falhas.

Resposta HTTP

Qualquer outro código de resposta que não seja um "200 OK" é considerado um erro, e um alarme será disparado. Note que - dependendo de sua conta no Pachube - restrições em requisições à API podem ser aplicadas. Veja http://community.pachube.com/api#rate_limit para detalhes.

Pontos publicados

A seção Pontos lista os pontos a serem publicados.A publicação de pontos pode ser habilitada () ou desabilitada (). Desabilitar um ponto permite que ele não seja utilizado sem necessidade de deletá-lo. Os valores Feed ID e Data stream ID determinam o fluxo de dados que será atualizado.

Para adicionar um ponto à lista de pontos, selecione-o na lista de pontos e clique no ícone . Para deletar um ponto existente, clique em seu ícone associado.
