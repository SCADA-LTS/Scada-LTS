A lista de data sources apresenta todos os data sources definidos no sistema, com os nomes, tipos, atributos de conexão e status de cada um. Os valores para Conexão são configurações que dependem do tipo de data source. O valor de Status determina se um data source está habilitado () ou desabilitado (). Clicar sobre o ícone alternará seu status.

Para adicionar um novo data source, selecione o tipo desejado na lista e clique sobre o ícone . Para editar um data source existente, clique no ícone associado ao data source. Para remover um data source, clique no ícone associado ao data source. Dependendo da quantidade de dados (data points) associados ao data source, o processo de exclusão pode demorar uma quantidade significativa de tempo. Certifique-se de que o processo terminou por completo antes de continuar.
