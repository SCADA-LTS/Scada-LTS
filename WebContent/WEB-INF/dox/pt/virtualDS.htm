O data source virtual é uma fonte de dados sem nenhum tipo de conexão com qualquer sistema externo. Seus dados são produzidos internamente pelo sistema de diferentes formas, como rândomica ou determinística, dependendo de como você configura seus data points.

Data sources virtuais podem ser úteis de diversas maneiras:

  • É uma maneira simples de validar se o sistema está funcionando corretamente.
  • Pode criar dados preditivos para lidar com seus dados reais.
  • Permite controle de equipamentos de formas mais complexas. Por exemplo, se a luz de sua casa é controlada pelo sistema, você pode usar um data point binário aleatório para controlar as luzes enquanto viaja, criando a impressão de que existem pessoas na casa.
  • Em qualquer lugar em que dados determinísticos ou aleatórios sejam úteis.

A configuração deste tipo de data source é simples. Só requer um nome e o período de tempo entre cada atualização dos dados. Por exemplo, se você escolher "2 minutos", então a cada 2 minutos os algoritmos que trocam os valores dos data points virtuais serão executados.
