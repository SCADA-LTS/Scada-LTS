Visualizações podem ser compartilhadas com outros usuário de maneira similar ao modo como são compartilhadas com usuários anônimos. Visualizações compartilhadas não podem ser editadas pelo usuário com qeum você está compartilhando (apesar de que este usu7ário tem o direito de se retirar da lista de compartilhamento). Para compartilhar uma visualização que você criou, selecione o usuário da caixa de seleção e clique no ícone . Então, selecione o tipo de acesso que este usuário deve ter, as opções são "read" (sem permissão de definir pontos) ou "set" (com permissão de definir pontos). Adicione tantos compartilhamentos à visualização quantos forem necessários. Para salvar, clique no botão "Save". Para remover um usuário da lista de compartilhamento, clique no ícone associado ao usuário.
