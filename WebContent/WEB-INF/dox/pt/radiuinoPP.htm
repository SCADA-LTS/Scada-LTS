Configuração

Todo datapoint Radiuino exige um Nome com qualquer descrição.

O Endereço do sensor indica qual o endereço do sensor que responderá pelos dados deste datapoint.

O Tipo de dado radiuino indica que tipo de dado está sendo recebido no pacote Radiuino.

O Indice do byte (pacote Radiuino) indica qual é o primeiro byte dentro do pacote Radiuino que se encontra os dados deste datapoint.

O Multiplicador permite multiplicar o valor recebido por um fator definido pelo usuário.

O Offset permite somar ao valor recebido um valor definido pelo usuário.

No item avançado temos outras opções que somente deverá ser utilizado por quem conhece a estrutura interna do Radiuino. Não utilize caso não conheça profundamente a estrutura Radiuino.

O Mapa de bytes no pacote de polling permite definir valores fixos dentro do pacote de polling.

O Mapa de bytes no pacote de requisição permite definir valores fixos dentro do pacote de envio de valores.
