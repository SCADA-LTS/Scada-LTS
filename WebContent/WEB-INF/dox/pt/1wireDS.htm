Descrição

O data source 1-wire é utilizado para coletar informaçã de uma rede local 1-wire, acessada por comunicação RS232 ou RS485 (requer um adaptador). É um data source do tipo polling.

O data source é capaz de tolerar reconfigurações da rede física. Quando são detectadas alterações na topografia de uma rede, uma atualização é feita. Se todos os pontos estiverem localizados após a leitura da rede, nenhum evento é levantado.

Configuração

Todo data source necessita de um Nome, que pode ser qualquer descrição. A seleçã da Porta oferece uma listagem de todas as portas seriais disponíis no hardware host, que corresponde a uma porta onde o 1-wire será conectado. Pesquisa na rede

Redes podem ser pesquisadas buscando por dispositivos utilizando a função pesquisa. Esta funcionalidade irá pesquisar através dos caminhos conectados por engates para localizar todo os equipamentos conectados na rede. Os resultados são mostrados em uma tabela como endereço e descrição do dispositivo. Para adicionar rapidamente um dispositivo como um ponto, clique no ícone . Um novo ponto será criado, com os atributos padrões do dispositivo.
