Descrição

Este datasource possibilita a leitura de dados provenientes de um OPC Server

Configuração

Todo datasource exige um Nome, que pode ter qualquer descrição.

O Host indica o endereço local do cliente.

O Domínio indica o domínio no qual o cliente se encontra. Obs.: O Cliente deve estar necessariamente em um domínio para operar com o OPC.

O Usuário indica o nome de usuário do Sistema Operacional.

A Senha indica a senha do nome de usuário do Sistema Operacional.

O Servidor mostra qual Servidor está diponível para os dados definidos.

O Período de Atualização indica o período de atualização dos dados do datasource.

O Método de Criação indica como as Tags OPC serão criadas no cliente. Tem duas opções: Listar Tags - onde todas as Tags do servidor serão listadas e Adicionar Tag - onde será adicionada uma Tag diponível do servidor
