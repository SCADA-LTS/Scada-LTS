Visão Geral

Uma watch list é uma lista dos data points nos quais você tem interesse em observar em conjunto. Os dados são atualizados automaticamente — você não precisa atualizar a página (F5). (Inclusive, o sistema opera de maneira mais eficientemente caso você não atualize).

Watch Lists Nomeadas

Um usuário pode ter múltiplas watch lists, porém só pode observar uma por vez. Para criar uma nova lista, clique no ícone . Novas listas são nomeadas como "(sem nome)" por padrão. Para renomear uma watch list, paire o cursor do mouse sobre o ícone para habilitar o painel de edição, entre com o novo nome e pressione Enter ou clique em "Salvar" para salvar. Para deletar, clique em . Cada usuário deve possuir no mínimo uma watch list, portanto o ícone para apagar só será habilitado se houver mais de uma lista.

Pontos em watch lists

Um data point pode pertencer a múltiplas listas, para que você possa criá-las tão específicas quanto precisar.

Os dados mostrados referentes a cada data point depende da configuração do mesmo. Data points que são setáveis (e que você possui a permissão para escrita) mostrarão o ícone (). Ao pairar o cursor do mouse sobre o ícone serão exibidas as opções para escrita. Data points que são associados a gráficos mostram o ícone (). Pairar o cursor do mouse sobre este ícone exibe o gráfico. Os gráficos, assim como os outros dados, também são atualizados automaticamente. O ícone de detalhes/configurações dos data points () levará você à pagina de detalhes para o ponto selecionado.

Para reordenar os data points, use os ícones: clicando em moverá o data point uma linha a baixo na lista, enquanto clicar em o moverá uma linha acima.

Para adicionar data points a sua lista corrente, clique sobre o nome do ponto na seção data points. Eles serão anexados no final da lista. Para removê-los, clique sobre o ícone (). Clicar sobre este ícone removerá o data point somente da lista corrente — não do sistema. Não confunda o ícone de remover o ponto com o ícone de remover a watch list no topo da página

Alteração dos valores

Os dados mostrados na watch list serão brevemente destacados quando houver alterações. O destaque permanece até o próximo ciclo de atualização. Os timestamps normalmente mudarão mais que os valores em si, pois a atualização de um ponto sempre envolve a atualização de seu timestamp, mas não necessariamente de seu valor.
