O Tipo de dado é a representação do valor no ScadaBR. Dados recebidos são automaticamente convertidos para o tipo de dado requerido.

Os campos Feed ID e Data stream ID são usados para identificar exclusivamente um fluxo de dados. Pedidos por múltiplos fluxos em um único feed são for multiple streams within a single feed are unificados por eficiência.

Para tipo de dado binário, o Valor 0 binário pode ser definido. Se o valor encontrado no conteúdo casar com ele, o ponto será atualizado com 0. Caso contrário, o ponto é atualizado com 1.

Para garantir que seu fluxo de dados existe, clique no ícone associado.
