Configuração

Todo datapoint exige um Nome com qualquer descrição.

O Tipo de dado indica que tipo de dado está sendo requisitado pela porta serial.

A Regex de Valor permite o usuário setar a sua regex de preferência. Note que já existe uma regex pré-configurada para Inteiro/Float para números positivos ou negativos.

O Timestamp é um checkbox que permite que o usuário escolha regex de timestamp pré-configuradas.
