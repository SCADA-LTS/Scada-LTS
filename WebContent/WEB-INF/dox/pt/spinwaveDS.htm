Descrição

O data source Spinwave é utilizado para obter dados de dispositivos Spinwave conectados a um gateway local, e acessível via comunicação RS232 ou RS485 (requer adaptador). Este data source recebe atualizações regulares dos dispositivos conectados ele não realiza polls.

Configuração

Todo data source requer um Nome, o qual pode ser qualquer descrição. A seleção Porta fornece uma lista de todas as portas seriais disponíveis na máquina host. Isto corresponde à porta na qual o gateway Spinwave está conectado. A Versão da mensagem depende do hardware que você está usando. Consulte a documentação de seu Spinwave para detalhes.

Endereços dos sensores são tipicamente encontrados usando a função de listener de sensores descrita abaixo, embora se você souber os endereços únicos de seu hardware você pode entrar com eles manualmente. Após entrar com o valor do endereço, clique no ícone para adicioná-lo à lista de endereços conhecidos.

Spinwave sensor listener

Uma vez o gateway Spinwave conectado ao host, um listener pode ser iniciado, o qual aguarda por mensagens. Tipicamente, dispositivos são configurados para enviar mensagens a cada 21 segundos, mas outras configurações são possíveis. Para adicionar o endereço de todos os dispositivos que foram ouvidos à lista de endereços de sensores, clique no botão "Adicionar sensores encontrados à lista".
