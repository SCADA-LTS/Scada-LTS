Esta página apresenta os controles para a edição das propriedades de um data point. O ícone oferece um link para a página, de leitura apenas, com os detalhes do data point, acessível apenas para os que foram salvos.

O Nome do data point é obrigatório e pode ser qualquer valor de texto. Este nome será usado como a identificação visual do mesmo por todo o sistema.

O nome do data source aparece como referência. O ícone é um link para a página de edição do data source, onde sua configuração pode ser encontrada e aparece como conveniência.
