Visão geral

O data source VMStat é utilizado para monitorar diversos aspectos relacionado a performance de um host *nix. Quando o data source é iniciado um um processo que executa o programa vmstat e a saída desse processo é monitorada.

Configuração

Todo data source necessita de um Name, que pode ser qualquer descrição. A configuração Poll seconds determina quão freqüente o processo vmstat irá retornar os dados coletados para o ScadaBR. A Output scale pode ser configurada para automaticamente ajustar os valores em bytes para escalas diferentes, incluindo bytes, k, KB, m e MB.
