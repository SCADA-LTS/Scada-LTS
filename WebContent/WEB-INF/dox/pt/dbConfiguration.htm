O ScadaBR permite no momento a utilização de dois Sistemas Gerenciadores de Banco de Dados (SGBD): Derby e MySQL.

O Derby é utilizado como padrão pelo sistema. Porém, caso seja necessário alguma modificação em sua configuração inicial, é necessário a modificação do arquivo localizado em: "ScadaBR*/WEB-INF/classes/env.properties" e "ScadaBR*/WEB-INF/classes/env.properties.derby".

Para utilizar o MySQL, é necessário inserir os dados necessários para a sua utilização correta. Dessa maneira, o arquivo necessário a ser modificado encontra-se em: "ScadaBR*/WEB-INF/classes/env.properties.mysql". Basicamente os seguintes parâmetros devem ser inseridos:

db.username: Nome do usuário.

db.password: Senha do usuário.

Feito as devidas modificações, é necessário escolher o MySQL como opção e clicar no ícone de salvar. Na sequência reinicie o Servidor de Aplicação(ex. Tomcat).

ScadaBR*: Localização do ScadaBR no servidor de aplicação(ex: "Pasta_Tomcat/webapps/ScadaBR").
