O OID representa tanto o identificador do objeto no host ou para equiparar a traps recebidas. O Data type é a representação do ScadaBR para o valor. Dados recebidos são automaticamente convertidos no tipo de dado necessário. O Binary 0 value é o valor para comparar para um ponto binário a ser atribuí ao valor 0. O Set type é o tipo de dados utilizado para definir valores no host. Um valor "(not settable)" indica que o ponto pode ser apenas lido. O campo Polling determina se devem ou não ser incluído o valor em requisiç&oteilde;es GET em cada ciclo de polling ou se valores são apenas recebidos via traps.
