Descrição

Um processador de gráficos é utilizado para proporcionar o estado histórico agregado de um ponto ao longo de um período de tempo ou de um número de valores. A informação agregada que pode ser providenciada depende do tipo de dado dos valores a serem processados. Assim, apenas os tipos que podem ser processados para o tipo de dado em quest&aatilde;o serão mostrados na seleção do Tipo.

Atenção! Processadores de gráficos dependem dos dados registrados. Se o ponto tiver um tipo de registro "Do not log", o processador de grá pode não exibir o gráfico conforme esperado. A exceção potencial é o gr´fico Tabela, que pode exibir valores armazenados em memória, mas apenas se estes valores estiverem disponíveis. Reiniciar o sistema elimina todos os valores de pontos armazenados em memória.

Tabela

Este processador de gráficos mostra um determinado número (o Limite) dos valores históricos mais recentes para o ponto. Valores são mostrados em uma tabela fornecendo o valor (processado como texto), o tempo relacionado ao valor e qualquer anotação aplicável. Este gráfico está disponível a todos os tipos de dados.

Gráficos estatísticos

Este gráfico mostra uma tabela de informações estatísticas calculadas a partir dos valores históricos do ponto para o Período de tempo determinado. Os dados reais mostrados dependem do tipo de dado do ponto. Este gráfico está disponível a todos os tipos de dados.

Binário and Multi-estado: Estes tipos de dados produzem uma lista de estatísticas do tipo "início e execução" para cada estado que o ponto experimentou durante o período. Cada ítem início e execução proporciona o número de vezes que o estado ("início") foi atingido, o tempo total em que o ponto esteve no estado ("execução") e o percentual do tempo total do período que o estado execuçãorepresenta.

Numérico: Pontos numéricos produzem os seguintes valores estatísticos para o período de tempo: valor m&icaute;nimo e o momento (tempo) em que ocorreu, valor m&acaute;ximo e o momento (tempo) em que ocorreu, média e soma (aplicável para contagem de pulsos).

Alfanumérico: O número de vezes que o ponto mudou de valor ao longo do tempo é fornecido..
