O IP do dispositivo remoto é o endereço IP aonde o dispositivo pode ser achado. A Porta remota do dispositivo é o número da porta em que o dispositivo escuta por requisições.

O Tipo de objeto é o tipo de objeto para o qual valores serão requisitados. O Número de instância de objeto é um identificador único do objeto/tipo para o qual valores serão requisitados. Pontos requisitam o valor atual de uma instância de objeto.

Usar subscrição de COV determina se o ScadaBR irá tentar subscrever a eventos COV para o objeto no dispositivo. Nem todos os objetos suportam subscrições COV e nem todos os dispositivos os suportam mesmo para objetos que normalmente suportariam.

Utilize Definir para determinar se a interface irá permitir que usuários definam valores para atualizar o objeto. Somente valores do tipo "output" podem ser definidos.

O campo Tipo de Dado determina qual tipo de dados do ScadaBR que o campo terá. Valores retornados da rede BACnet ser&aatilde;o automaticamente convertidos para o tipo definido.
