Configuration

Aujourd'hui, 3 types de données sont pris en charge : Point unique, double mesure ponctuelle et normalisée.

Commandes

Pour envoyer des commandes, vous pouvez spécifier un offset, à ajouter à l'adresse de l'objet lorsque la commande est envoyée. Par exemple : lorsque en écrivant l'adresse 2 d'un objet avec le décalage 2000, la commande sera envoyée pour s'opposer avec l'adresse 2002.
