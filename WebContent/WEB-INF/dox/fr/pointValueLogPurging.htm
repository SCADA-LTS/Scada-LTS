La fonction Purger maintenant vous permet de purger facilement les valeurs d'un points. Les valeurs purgées sont irrécupérables après cette opération. Sélectionnez Purger toutes les données antérieures à ou cliquez sur Purger toutes les données et cliquez sur le bouton Purger maintenant. La suppression est réalisée de manière asynchrone, vous pouvez donc allez sur une autre page pendant cette opération mais dans ce cas vous ne recevrez pas de confirmation de la purge.
