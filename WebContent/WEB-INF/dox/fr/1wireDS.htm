Vue d'ensemble

Une source de données 1-wire est utilisée pour collecter les données d'un réseau local de communication 1-wire accessible via RS232 ou RS485 (nécessite un adaptateur). C'est une source de données de type polling.

La source de données supporte les changements physique de configuration du réseau. Pour chaque modification topographique de réseau détectée, un nouveau scan est réalisé . Si tous les points sont encore localisables après le scan, aucun évènement n'est remonté.

Configuration

Chaque source de données exige un Nom, librement choisi. La sélection du Port fournit la liste de tous les ports série disponibles du matériel Hôte. Cela correspond au port sur lequel le 1-wire est connecté. Mise à jour détermine la fréquence d'interrogation du réseau.

Scan du réseau

Les réseaux peuvent être scannées pour rechercher tous les périphériques présents (si le matériel le permet) . Cette fonction va localiser à travers les mailles couplées tous les équipement du réseau. Les résultats seront représentés dans un tableau comprenant l'adresse et la description de l'appareil, pour ajouter rapidement un dispositif en tant que point cliquez sur l'icône . Un nouveau point sera crée avec les attributs par défaut de l'appareil.
