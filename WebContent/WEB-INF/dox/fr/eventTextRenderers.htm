Overview

An event text renderer's purpose is to give more specific information what happened when event is triggered. Each type of renderer has a text field where user can define message which is related to specific point value/state. Event text renderer value will be included in message from each type of detector (Change, BinaryState, HighLimit, etc.) and will be shown on watch list, graphical view, mail/SMS notification - in all places where event detector message is displayed. The ways to render depend upon the data type of data point. As such, only the renderer types that apply to the current point's data type are listed in the Type selection.

Binary

The Binary renderer allows the user to define an event text label to describe each of the two binary states. Any labels can be provided.

Numeric

The Range renderer allows the user to define a textual values to represent event states in various numeric from/to ranges.

Multistate

The Multistate renderer allows the user to define textual values to represent event states in various point states.
