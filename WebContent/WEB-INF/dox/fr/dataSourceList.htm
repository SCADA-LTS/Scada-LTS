La liste des sources de données affiche toutes les sources de données définies dans le système, nom , type, parametres de connexion, et leurs statuts. Les paramètres Connexion affichent les configurations dépendantes du type de la source de données. Les Statuts déterminent si la source de donnée est activée ( ) ou désactivée ( ). En cliquant sur l'icone statut, vous permuttez le statut de la source de données.

Pour ajouter une nouvelle source, sélectionnez le type de source de données dans la liste et cliquez sur l'icone en haut de la liste. Pour éditer une source de données existante, cliquez sur l'icone associée la source de données. Pour supprimer une source de données, cliquez sur l'icone associée à la source de données. La durée de suppresion d'une source de données dépend du nombre d'enregistrements de cette base; Il faut etre certain de la fin du processus de suppression avant de continuer.

Les icones et affichent ou cachent les points de la source de donnée. Comme les source de données, les points peuvent etre activés ( ) ou désactivés ( ).
