Configuration

Every datasource requires one Name, that may contain any description.

The Source Address is the master address, or, the datasource's to be registered address and the Slave Address is the remote device address to which you wish to communicate.

The RBE Poll Period corresponds to the time interval upon which the system makes requisitions of classes 1, 2 an 3 (events) to the slave device. The Static Poll Period corresponds to the time interval upon which the system makes requisitions of class 0 to the device and is relative to the RBE polling period, so, if RBE period = 2s and Static Poll Period = 30, the datasource will require class 0 data every 60 seconds to the device.
