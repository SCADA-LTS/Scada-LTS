Cette section fournit la liste de tous les points du système auxquels vous avez accès. Votre accès dépend de vos paramètres de la zone utilisateurs (). Contactez votre administrateur s'il vous manque des points utiles. Si vous ne voyez aucun point (pour une nouvelle installation par exemple ... bienvenue!), vous devez les créer en créant des sources de données ().

Les points sont classés dans une organisation, ou arborescence, tel que défini dans la zone Organisation des points (). L'organisation est totalement arbitraire, et partagée par tous les utilisateurs. Vous devez être administrateur pour modifier l'organisation des points.

Pour ajouter un point de l'organistaion, cliquez sur son nom. L'icône de flêche () disparaîtra pour indiquer que le point fait parti de la liste de surveillance courante.
