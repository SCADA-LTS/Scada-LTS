Email settings are used for all emailing activities within the application instance.

SMTP host is the name of the host where your MTA is running. If your host requires authentication, check the Use authorization box, and fill in the Username and Password fields. Check the Enable TLS checkbox if your server requires it.

The 'From' address is the address that will be used as the from address on each email. The 'From' name is an optional field with which to specify an alias—or 'personal' name—to display as the email sender.

Use the Content type select box to choose the type of content you wish to have in the Mango emails. By default HTML and text is selected, which causes Mango to create a multi-part MIME email with both "text/html" and "text/text" parts. If HTML only is chosen the email may still be a multi-part if there are attachments or in-line content, but will be otherwise be single part. If Text only is chosen, the email will always be a simple single part. Note: scheduled report emails are always sent as HTML-only due to the content they include.

When you are finished with your email settings, you can test them by clicking the icon. This will attempt to send an email to your email address using the settings you have entered.
