Overview

The 1-wire data source is used to gather data from a local 1-wire network, accessible via RS232 or RS485 (requires adapter) communication. This is a polling data source.

The data source is able to tolerate physical network reconfigurations. When changes to the topography of a network are detected, and automatic rescan is done. If all points are still located following the scan, no event is raised.

Configuration

Every data source requires a Name, which can be any description. The Port selection provides a list of all available serial ports on the host hardware. This corresponds to the port into which the 1-wired is connected. The Update period determines how often the network is polled for data.

Network scanning

Networks can be scanned for devices using the scan utility. This feature will scan through paths connected by couplers to locate all equipment on the network. The results are displayed in a table providing the 1-wire address and a description of the device. To quickly add a device as a point, click the associated icon. A new point will be created with the device's attributes defaulted.
