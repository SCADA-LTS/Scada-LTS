The Data type is the Mango representation of the value. Received data is automatically converted to the required data type.

The Feed ID and Data stream ID are used to uniquely identify a data stream. Requests for multiple streams within a single feed are consolidated for efficiency.

For the binary data type, the Binary 0 value can be defined. If the value found in the content matches it, the point will be updated with 0. Otherwise, the point is updated with 1.

To ensure that your data stream exists, click the associated icon.
