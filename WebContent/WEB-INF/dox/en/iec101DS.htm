Configuration

Every datasource requires a Name, that can contain any description.

All initial parameters depend on the slave device configurations and should be configured correctly..

The Update period corresponds the time interval upon which the system makes calls 2 and, if necessary, class 1 requisitions to the slave device.

The GI relative period corresponds to the time interval upon which the system makes the requisition of one General Interrogation to the device and is relative to the update period defined beforehand. So, if the update period = 2s and the GI period = 30, the datasource sends one General Interrogation every 60 seconds to the device.
